# MoinMoin

::: {#content dir="ltr" lang="en"}
MoinMoin is a Wiki software implemented in [Python](http://moinmo.in/Python "MoinMoin"){.interwiki} and distributed as Free Software under [GNU GPL v2+](http://moinmo.in/GPL "MoinMoin"){.interwiki} license.

The name \"MoinMoin\" is a common German slang expression (see [our \"MoinMoin\" etymology page](http://moinmo.in/MoinMoinEtymology "MoinMoin"){.interwiki} for more about that).

For more information about MoinMoin, please see the project wiki at [http://moinmo.in/](http://moinmo.in/){.http}.
:::
