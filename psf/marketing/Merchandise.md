## Ideas for Python Merchandise 

- Python t-shirts
- Python caps
- Python coffee mugs
- Python pens
- Python USB sticks (possibly with Python material preloaded)
- sets with Python flyers and stickers

## Ideas for PSF Merchandise 

- PSF t-shirts
- PSF caps
- PSF coffee mugs
- PSF pens
- PSF USB sticks (possibly with Python material preloaded)
- PSF branded Raspberry Pis (just the Pis, with the branded OS image on the python.org website)
- sets with PSF flyers and stickers

## Ideas for Member Giveaways 

All of the above, plus\...

- gift coupons for bookshops (e.g. for O\'Reilly and bookshops)

- Python e-books that we purchase in bulk (or get from O\'Reilly or one of the other sponsors)

- discount coupons for Python conferences (e.g. [PyCon](./PyCon.html) NA and [EuroPython](./EuroPython.html))

- sets with PSF/Python flyers and stickers
