## Purpose & Common Goals {#Purpose_.26_Common_Goals}

The purpose of this workgroup is to discuss and vote on creation/dissolution of PSF working groups.

## Active Time {#Active_Time}

This workgroup will be active as long as the PSF uses working groups.

## Core Values & Internal Government {#Core_Values_.26_Internal_Government}

The core values for this group are:

- Respect
- Support
- Positive Atmosphere
- Constructiveness
- Integrity

The workgroup adopts the PSF Code of Conduct ([http://psf2013.election.name/evote/default/results/40](http://psf2013.election.name/evote/default/results/40){.http}). Any actions (found by a majority) made against the principles in the Code of Conduct will result in the acting person being removed from the workgroup.

## Rules & Decision Making Procedures {#Rules_.26_Decision_Making_Procedures}

Ideas and implementations will be proposed to the group and changes will be decided by a majority rule.

## Communication Plan {#Communication_Plan}

The team will communicate via mailing list. If needed, conference calls can be set up.
