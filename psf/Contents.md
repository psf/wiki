# Python Software Foundation (PSF) Wiki

**Please note:** The wiki pages in this wiki default to *world readable*. Unlike in previous years, the wiki is no longer private. You do need to be in the [MembersGroup](MembersGroup){.reference .external} to make changes. For this, you will need to create an account in this wiki (click on \"Login\" on the left). Please contact the staff for details ([psf-staff@python.org](mailto:psf-staff@python.org){.mailto .reference .external}).
