# PSF Marketing Workgroup Charter 

## Purpose & Common Goals 

The purpose of this workgroup is to create marketing strategies, material and kits for use by the PSF and the Python community at large.

## Active Time 

This workgroup will be active indefinitely.

## Core Values & Internal Governance 

The workgroup adopts the PSF Code of Conduct. Any actions (found by a majority) made against the principles in the Code of Conduct will result in the acting person being removed from the workgroup.

## Rules & Decision Making Procedures 

Ideas and implementations will be proposed to the group and changes will be decided by a majority rule.

## Communication Plan 

The team will communicate via mailing list. If needed, conference calls can be set up. Agenda documents will be kept in Google Docs.

- [psf-marketing-wg@python.org](mailto:psf-marketing-wg@python.org)
