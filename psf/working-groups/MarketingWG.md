# PSF Marketing Working Group 

*Note:* When creating new wiki pages related to this group, please use wiki subpages, e.g. MarketingWG/Charter.

## Agenda 

The marketing workgroup will strive to:

- Create, store and distribute marketing material, e.g. logos, images, flyers, posters, stickers, handouts, brochures, etc. both in electronic and printed form
- Create, store and distribute marketing support material, e.g. banners, displays, table covers, swag, etc.
- Create, store and distribute Python merchandise, e.g. t-shirts, caps, mugs, etc.
- Create and run an online Python merchandise shop system
- Design and create a Python press kit for press people to easily learn about Python
- Design and create a Python advocacy kit for community members to use in teaching other people about Python
- Curate a regularly updated set of Python success stories on python.org
- \... more \...
