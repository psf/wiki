The Python Wikipediabot Framework, also called pywikipedia is a set of bot scripts to do things on Wikipedia or other wikis using [MediaWiki](http:\\www.mediawiki.org). It has been written by several people using the language Python. It is available form [SourceForge](http://sourceforge.net/projects/pywikipediabot/) or via SVN at [http://svn.wikimedia.org/svnroot/pywikipedia/trunk/pywikipedia/](http://svn.wikimedia.org/svnroot/pywikipedia/trunk/pywikipedia/) For help go to [#pywikipediabot](irc://irc.freenode.net/pywikipediabot) @ freenode server or use the [pywikipediabot mailing list](http://lists.wikimedia.org/mailman/listinfo/Pywikipedia-l). A guide to pywikipedia can be found [here](http://meta.wikimedia.org/wiki/Using_the_python_wikipediabot). There is also a wiki where is possible to test your scripts, [Botwiki](http://botwiki.sno.cc/wiki/Main_Page).
