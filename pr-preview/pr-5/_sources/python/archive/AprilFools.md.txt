The Python community has a long history of April Fool\'s postings to [news:comp.lang.python](news:comp.lang.python) (a.k.a. [mailto:python-list@python.org](mailto:python-list@python.org)). We\'ve collected references to a number of them here. If you know of more, please update the page.
