This page lists Python books written in Dutch.

Deze pagina geeft een lijst van Nederlandstalige boeken over Python. Aangezien de meeste Nederlanders het Engels goed beheersen, zijn er niet veel Nederlandstalige boeken beschikbaar.

------------------------------------------------------------------------

[De Programmeursleerling: Leren coderen met Python 3](http://www.spronck.net/pythonbook/dutchindex.xhtml)

Pieter Spronck

\~400 pagina\'s (juli 2016)

**Overzicht:**

\"De Programmeursleerling\" is een Python 3 cursusboek voor beginners, gericht op middelbare scholieren en studenten voor wie programmeren nieuw is. In tegenstelling tot veel andere boeken die Python doceren, veronderstelt dit boek geen voorkennis bij de studenten op programmeergebied, en bevat het veel opgaves waarmee de studenten kunnen oefenen.

Het boek behandelt alle imperatieve en object georiënteerde aspecten van Python, waaronder:

- Installeren, schrijven, en uitvoeren van Python en Python programma\'s

- Expressies, variabelen, condities, loops en functies

- Strings, tuples, lists, dictionaries en sets

- Tekstbestanden, binaire bestanden en exceptions

- Object oriëntatie, operator overloading, overerving, iteratoren en generatoren

- Reguliere expressies

Het boek besteedt ook tijd aan het ontwerpen van algoritmes en het schrijven van effectieve code.

Het boek is gratis beschikbaar als PDF.
