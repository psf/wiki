# VISA instrumentation 

- pyvisa ([http://sourceforge.net/projects/pyvisa](http://sourceforge.net/projects/pyvisa)) is package with bindings to the [\"Virtual Instrument Software Architecture\"](http://en.wikipedia.org/wiki/Virtual_Instrument_Software_Architecture) library that can be used to control measurement devices and test equipment via GPIB, RS232, or USB interfaces.
