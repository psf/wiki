[stpconvert.py](http://www.68k.org/~michael/macosx/stpconvert.py) uses [../AppscriptModule](AppscriptModule) and [Soundtrack Pro](http://www.apple.com/finalcutstudio/soundtrackpro/) to convert a file to a different file format, sample format, or sample rate.

[stpfindfix.py](http://www.68k.org/~michael/macosx/stpfindfix.py) uses [../AppscriptModule](AppscriptModule) and [Soundtrack Pro](http://www.apple.com/finalcutstudio/soundtrackpro/) to detect and repair common audio defects such as clicks and pops, hum, and phase misalignment.
