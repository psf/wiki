# Books & Publications

A collection of Python book lists, organized by topic and skill level. These pages were community-maintained catalogs back when finding a good Python book required more than a search engine. Many of the specific titles listed are dated, but the pages give a sense of the Python publishing ecosystem over the years.

## General Catalogs

- [PythonBooks](PythonBooks) — the main book directory, linking to categorized lists and specialized topics
- [Any good Python book?](Any%20good%20Python%20book_) — community recommendations and discussion about picking a Python book
- [PythonBooks for a list.](PythonBooks%20for%20a%20list.) — additional book listing page

## By Skill Level

- [IntroductoryBooks](IntroductoryBooks) — gentle overviews of the language for beginners
- [AdvancedBooks](AdvancedBooks) — books for experienced developers who want depth
- [ReferenceBooks](ReferenceBooks) — dense reference manuals that pack a lot of information into a small space

## By Topic

- [NetworkProgrammingBooks](NetworkProgrammingBooks) — books focused on network programming with Python
- [SystemAdministrationBooks](SystemAdministrationBooks) — books on using Python for sysadmin tasks

## By Language & Platform
