The Python Software Foundation aims to serve as an \"incubator\" organization for smaller Python projects in Google Code-In 2011. Toward this end our example tasks list is available on several subpages managed by each project.

[PySoy](http://www.pysoy.org/wiki/CodeIn/2011)

[SymPy](http://code.google.com/p/sympy/issues/list?q=CodeInCategory)

If accepted, additional Python-based projects will be encouraged to apply. Write Arc Riley \<[arcriley@gmail.com](mailto:arcriley@gmail.com)\> for details.
