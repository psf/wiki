## Jussi Kasurinen 

Email: [jussi.kasurinen@lut.fi](mailto:jussi.kasurinen@lut.fi)

LUT Home: [Entry](http://www2.lut.fi/~jukasuri/)

A researcher with the Lappeenranta University of Technology\'s Technology Business Research Center, who is currently working on a research project on software engineering and testing practices while compiling he\'s Doctoral Thesis.

Research interests include but are not restricted to computer science education, virtual learning environments, e-learning, user interface design, user behavior analysis and data mining.

Contributes to Python community by writing introductory material in Finnish language and promoting Python as a first programming language in high schools and colleges. Future prospects include Finnish Python-blog and a personal project on Python 3.
