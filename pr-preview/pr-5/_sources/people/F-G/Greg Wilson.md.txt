- Project lead at [Software Carpentry](http://software-carpentry.org), an open source course on basic computational skills for scientists and engineers.

- Regularly supervises [Google Summer of Code](http://code.google.com/soc/) projects and [other student programming projects](http://ucosp.ca).

- Author of *[Data Crunching](http://www.amazon.com/Data-Crunching-Everyday-Problems-Python/dp/0974514071)*, co-author of *[Practical Programming](http://www.amazon.com/Practical-Programming-Introduction-Pragmatic-Programmers/dp/1934356271)* (a CS-1 textbook using Python), co-editor of *[Beautiful Code](http://www.amazon.com/Beautiful-Code-Leading-Programmers-Practice/dp/0596510047)* (winner of a 2008 Jolt Award), and currently working on books on empirical software engineering (for O\'Reilly) and the architecture of open source applications (for Pragmatic).

- Co-author of [PEP 218](http://www.python.org/dev/peps/pep-0218/): Adding a Built-In Set Object Type (and still disappointed that you have to write \'set()\' instead of \'{/}\' for the empty set).

- Personal site at [http://www.third-bit.com](http://www.third-bit.com).

- Best contact point is [gvwilson@third-bit.com](mailto:gvwilson@third-bit.com).
