## Standalone mode - Libraries included 

Jython has now a standalone mode, where the core part is all included in one single `jython.jar`{.backtick} file. This has two advantages:

- simpler starting: `java -jar jython.jar`{.backtick}

- simpler embedding: just put `jython.jar`{.backtick} onto your classpath. No setting of `python.home`{.backtick} nor `python.path`{.backtick} required.

This really adds a Plug & Play feature to Jython.

There is a preview available: [snapshot build](http://homepage.sunrise.ch/mysunrise/ohumbel/2837/jython_snapshot.jar), and an overview of the [test results](http://homepage.sunrise.ch/mysunrise/ohumbel/)

One Open issue: the cachedir problem still remains. If the directory containing `jython.jar`{.backtick} is not writable, no cachedir will help importing java classes. This is a well known problem also persistent in non-standalone mode. We try to improve this in standalone mode.
