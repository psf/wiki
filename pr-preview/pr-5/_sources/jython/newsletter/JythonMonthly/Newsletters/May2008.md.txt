## New Jython Book 

Take a look at the new Jython book by author Nishio Hirokazu. This book is written in Japanese, and it aims to introduce Jython to Java programmers. It focuses on Python grammer and the differences between Java and Python. Examples in this book include tests using [JyConsole](./JyConsole.html), Doctest, and interactive MIDI controlling.

Take a look at the book site: [http://www.nishiohirokazu.org/jybook/](http://www.nishiohirokazu.org/jybook/)

Amazon Link: [http://www.amazon.co.jp/dp/4839922829/](http://www.amazon.co.jp/dp/4839922829/)
