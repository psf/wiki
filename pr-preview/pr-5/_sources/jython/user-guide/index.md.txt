# User Guide

Practical guidance for working with Jython day to day. These pages cover the interactive console, servlets, application servers, and general usage patterns. Several of the user guide pages here are different versions written over the years -- the content overlaps, but each version has its own flavor and emphasis.

## User Guides

- [Jython User Guide](Jython%20User%20Guide) -- A comprehensive guide to using Jython
- [JythonUserGuide](JythonUserGuide) -- Another edition of the Jython user guide
- [UserGuide](UserGuide) -- General-purpose Jython usage documentation
- [UserGuide11](UserGuide11) -- User guide specific to Jython 1.1
- [NewUsersGuide](NewUsersGuide) -- A guide aimed at people picking up Jython for the first time

## Console & Environment

- [ConsoleChoices](ConsoleChoices) -- How Jython's interactive console works, JLine integration, and how to switch between console backends
- [ReadlineSetup](ReadlineSetup) -- Configuring readline and tab-completion support in the Jython interpreter

## Servers & Deployment

- [PyServlet](PyServlet) -- Using PyServlet to write Java Servlets in Jython source files
- [WebSphere](WebSphere) -- Running Jython inside IBM WebSphere Application Server, which ships with Jython as its admin scripting language
- [SimpleApp](SimpleApp) -- The Apache Derby SimpleApp sample, ported to show Jython in a database context
