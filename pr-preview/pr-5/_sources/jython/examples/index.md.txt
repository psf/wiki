# Examples

Working code samples that show Jython in action. Most of these demonstrate Java-Python interop in concrete ways -- building Swing GUIs, talking to databases, calling Java libraries from Python, and the other direction too. Good for learning by reading and modifying real code.

## Swing & GUI

- [SwingExamples](SwingExamples) -- A collection of Swing GUI examples written in Jython
- [ADB SwingExamples](ADB%20SwingExamples) -- Additional Swing examples contributed by ADB
- [SwingSampler](SwingSampler) -- A sampler application demonstrating various Swing components
- [SwingWorker](SwingWorker) -- Using SwingWorker for background tasks in Jython GUIs
- [JtreeExample](JtreeExample) -- Building a JTree widget in Jython
- [TabbedExample](TabbedExample) -- A tabbed pane interface example
- [JythonNutpad](JythonNutpad) -- A Notepad-like text editor built with Jython and Swing

## Database

- [DatabaseExamples](DatabaseExamples) -- Database access patterns from Jython
- [ApacheDerby](ApacheDerby) -- Connecting to and using Apache Derby from Jython
- [RSMD.py](RSMD.py) -- ResultSetMetaData utility script for database introspection

## Java Integration

- [JythonClassesInJava](JythonClassesInJava) -- How to use Jython-defined classes from Java code
- [JythonModulesInJava](JythonModulesInJava) -- How to import and call Jython modules from Java
- [Log4jExample](Log4jExample) -- Using Apache Log4j for logging from Jython
- [PoiExample](PoiExample) -- Reading and writing Excel files with Apache POI from Jython
- [EasterJar](EasterJar) -- Packaging a Jython application into a JAR file

## Other

- [CoreJythonExamples](CoreJythonExamples) -- Basic Jython language examples
- [XmlRelatedExamples](XmlRelatedExamples) -- Working with XML parsing and processing in Jython
- [OtherExamples](OtherExamples) -- Miscellaneous examples that don't fit neatly into another category
