## Special Interest Groups 

Many Python projects are organized around Special Interest Groups (SIGs). Each SIG tends to have a mailing list, and works on sub-projects related to Python. There are SIGs for database development, distribution and packaging, the email package, etc. Below are a list of SIGs along with links to their mailing list information and sub-pages used to coordinate development.

::: {}
  --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- ------------------------------------------------------------------------------------------------------------
  **SIG name**                                                                                                                                                                                  **Description**
  [Distutils SIG](../packaging/Distutils) [(subscribe)](http://mail.python.org/mailman/listinfo/distutils-sig) [(archives)](http://mail.python.org/pipermail/distutils-sig/)            Works on packaging and distribution, Cheeseshop (PyPI), and related technologies
  [Email SIG](../archive/Email%20SIG) [(subscribe)](http://mail.python.org/mailman/listinfo/email-sig) [(archives)](http://mail.python.org/pipermail/email-sig/)                     Works on developing Python\'s [email package](http://www.python.org/doc/current/library/email.html)
  [Stdlib SIG](./Stdlib(20)SIG.html) [(subscribe)](http://mail.python.org/mailman/listinfo/stdlib-sig) [(archives)](http://mail.python.org/pipermail/stdlib-sig/)   Work on improving the test suite and standard library for different implementations of Python
  --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- ------------------------------------------------------------------------------------------------------------
:::
