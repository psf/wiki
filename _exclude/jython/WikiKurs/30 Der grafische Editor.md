# Der grafische Editor 

Alternativ zur Eingabe von Wiki-Markup per einfachem Text-Editor bietet [MoinMoin](MoinMoin) auch einen grafischen WYSIWYG-Editor an, der von der Oberfläche und Bedienung an normale Textverarbeitungen erinnert.

Technische Voraussetzung: Browser Mozilla/Firefox/Netscape oder IE in aktueller Version + Javascript aktiviert.

Welcher Editor angeboten bzw. persönlich bevorzugt wird, kann man in den [BenutzerEinstellungen](./WikiKurs(2f)30(20)Der(20)grafische(20)Editor.html?action=userprefs) einstellen. Man kann im Editor auch jeweils in den anderen Editor umschalten (Text-Modus vs. GUI-Modus).

Vorteile grafischer Editor:

- hübscher
- einfacher

Nachteile grafischer Editor:

- lädt langsamer
- weniger flexibel
- Implementierung ist noch nicht fehlerfrei
