# PyOhio2008/publicize

::: {#content dir="ltr" lang="en"}
Materials to help you (yes YOU) spread the word about [PyOhio](PyOhio).

Formal announcement:

------------------------------------------------------------------------

- [PyOhio](PyOhio), the first annual Python programming miniconference for Ohio and surrounding areas, will take place Saturday, July 26, in Columbus, OH.

The conference is free of charge. It will include scheduled talks, Lightning Talks, and unconference-style Open Spaces.

To get more information or to volunteer, see [http://pyohio.org](http://pyohio.org){.http}.
