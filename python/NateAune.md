# NateAune

::: {#content dir="ltr" lang="en"}
Nate Aune is founder and president of [Jazkarta, Inc.](http://www.jazkarta.com){.http} a Boston-based consulting and solutions provider specializing in Python-based web applications and Plone CMS implementations.

Nate has given talks at [EuroPython](EuroPython), [PyCon](PyCon), [PyConBrasil](./PyConBrasil.html){.nonexistent} and many Plone conferences. He is also the organizer of the [Boston Django User Group](http://django.meetup.com/6/){.http}, [Boston Python User Group](http://python.meetup.com/181/){.http} and the [Boston Plone User Group](http://bostonplone.org){.http}.

Nate blogs at [http://nateaune.com](http://nateaune.com){.http} and [http://blog.jazkarta.com](http://blog.jazkarta.com){.http}.
