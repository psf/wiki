# PythonBarCamp

::: {#content dir="ltr" lang="en"}
[BarCamps](./BarCamps.html){.nonexistent} are loosely structured conferences (also known as \'unconferences\'), usually held for one or two days. See [http://www.barcamp.org/](http://www.barcamp.org/){.http} for more discussion of this approach to conferences.

- [/DC](./PythonBarCamp(2f)DC.html): A Washington DC-area Python [BarCamp](./BarCamp.html){.nonexistent} being planned for October or November 2008.

- [Texas Python Unconference](http://pycamp.python.org/Texas/HomePage){.http}, held on September 15-16 2007.

- [PyOhio](PyOhio): a daylong miniconference being planned for midsummer in Columbus, Ohio.
:::
