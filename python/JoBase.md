# JoBase {#JoBase-1}

JoBase is a fast Python game library for beginner coders. It is written pure C and uses the OpenGL engine. JoBase has integrated physics and can also run in your browser! Please visit the official website at [jobase.org](https://jobase.org){.https}.

Learn to code with the fastest Python tools.
