# Python Discussions {#Python_Discussions}

If you want to discuss Python online, these are a few resources you can tap into:

- [Python Discussions](https://discuss.python.org/){.https} - One of the larger Python forum and a place to meet lots of Python experts in the field. Run by the Python Software Foundation; some channels have superceded historical mailing lists. Uses the Discourse platform, so also called \"Python Discourse\".

  - [PythonDiscourseIdeas](PythonDiscourseIdeas) - Wiki page for managing ideas discussed on Python Discourse.

- [Python Forum](https://python-forum.io/){.https} - An independent forum using English as main language.

- [Python-Forum.de](https://www.python-forum.de/){.https} - Another independent forum using German as main language.

- [Python on Reddit](https://www.reddit.com/r/Python/){.https} - English subreddit dedicated to Python discussions.

- [Learn Python on Reddit](https://www.reddit.com/r/learnpython/){.https} - English subreddit specifically for newbies

- [Python Discord](https://discord.com/invite/python){.https} - active Discord server for Python users.
