# EtienneRobillard/LibAuthKit

::: {#content dir="ltr" lang="en"}
### Synopsis {#Synopsis}

Unofficial branch/port of AuthKit.

Source code is included in the [django-hotsauce](https://www.isotopesoftware.ca/software/django-hotsauce/){.https} distribution. (extras/libauthkit)

### Features {#Features}

- OAuth2 framework support (requires oauthlib)
- Schevo database support (requires libschevo)
- HTTP cookie authentication/authorization (experimental)
