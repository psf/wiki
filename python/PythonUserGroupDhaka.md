# PythonUserGroupDhaka

::: {#content dir="ltr" lang="en"}
Python User Group Dhaka was started in February 2025 by [Tamal Anwar Chowdhury](TamalChowdhury) and many other Python enthusiasts. The group runs monthly meetups every 15th of a calendar month. The group\'s main communication channel is [Facebook Group](https://www.facebook.com/groups/pythondhaka){.https}.

## First Meetup {#First_Meetup}

The first meetup happened on February 15th, 2025 at Chandrima Udyan in Dhaka, Bangladesh.

Total attendees: 5
