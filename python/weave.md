# weave

::: {#content dir="ltr" lang="en"}
The weave package allows the inclusion of C/C++ within Python code. It is part of the larger [SciPy](SciPy) package ([http://www.scipy.org](http://www.scipy.org){.http}), but is also available as a stand alone package ([https://pypi.python.org/pypi/weave](https://pypi.python.org/pypi/weave){.https}) so that more people can try it out.

The GPL\'d [AsynCluster](http://foss.eepatents.com/trac/AsynCluster/wiki){.http} project contains a package *svpmc*, which has a module [svpmc.weave](http://foss.tellectual.com/trac/AsynCluster/browser/projects/AsynCluster/trunk/svpmc/weave.py){.http}. The *Weaver* class in that module provides a convenient API for running C/C++ code with *weave.inline*. See the [svpmc.sample](http://foss.tellectual.com/trac/AsynCluster/browser/projects/AsynCluster/trunk/svpmc/sample.py){.http} module and its companion [C code file](http://foss.tellectual.com/trac/AsynCluster/browser/projects/AsynCluster/trunk/svpmc/sample.c){.http} for an example of usage.
