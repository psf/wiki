# PyFit

::: {#content dir="ltr" lang="en"}
The Python port of the [FIT](http://fit.c2.com/){.http} acceptance test framework. Contains also the python port of the [FitLibrary](http://fitlibrary.sourceforge.net){.http}, an extension of the FIT framework.

- \- [http://pypi.python.org/pypi/PyFIT/0.8a2](http://pypi.python.org/pypi/PyFIT/0.8a2){.http}\
  - [http://fitnesse.org/FitServers.PythonFit](http://fitnesse.org/FitServers.PythonFit){.http}
:::
