# PythonWikiEngines

::: {#content dir="ltr" lang="en"}
**Here is a list of known Python-based [WikiEngines](http://c2.com/cgi/wiki?WikiEngines "Wiki"){.interwiki} sorted by status and alphabetical order.**

## Active Wikis {#Active_Wikis}

- [MoinMoin](MoinMoin) \-- an evolution over [PikiPiki](http://c2.com/cgi/wiki?PikiPiki "Wiki"){.interwiki} that you\'re using here - multiple ways to run, plugin-based architecture with lots of plugins and large community, GPL2. (2022)

- [Pwic.wiki](https://pwic.wiki){.https} \-- extendable wiki integrated with Markdown, SQLite and [OpenDocument](./OpenDocument.html){.nonexistent} Text, AGPL3. (2022)

- [SahrisWiki](http://sahriswiki.org){.http} \-- Mercurial backend support, plugins, MIT licensed. (2016)

- [Sphene Community Tools](http://sct.sphene.net){.http} \-- Django based wiki and forum applications. (2018)

- [Trac](http://trac.edgewall.org){.http} \-- an enhanced wiki, issue tracking and source code browser system. Supports multiple markup formats including [MoinMoin](MoinMoin), [reStructuredText](reStructuredText), html, and [textile](http://www.textism.com/tools/textile/){.http}. Runs as [FastCGI](http://fastcgi.com){.http}, WSGI or standalone. Pluggable architecture. Modified BSD License. (2022)

- [zim-wiki](http://www.zim-wiki.org){.http} - WYSIWIG personal/desktop wiki with index, links, image inclusion, checkboxes and bulleted lists, equations, graphviz; calendar, [ToDo](./ToDo.html){.nonexistent}; prints to clean html. Customizable, all-python over gtk2, inbuilt-server, uses filesystem as db; debian packaged. Documentation. GPL2. (2021)

## Unsupported Wikis {#Unsupported_Wikis}

The following projects are available but no more maintained:

- [Luminotes](https://github.com/osborne6/luminotes){.https} \-- a WYSIWYG personal wiki notebook for organizing your notes and ideas. (2014)

- [Markdoc](https://github.com/zacharyvoase/markdoc){.https} \-- a lightweight Markdown-based Wiki system, public domain. (Claimed as inactive)

- [Pyle](https://leastfixedpoint.com/tonyg/kcbbs/projects/pyle.html){.https} \-- GPL wiki with multiple VCS backends, email notifications, edit by FTP and lot of other features. (2012)

- [Pwyky](http://infomesh.net/pwyky/){.http} \-- [WyPy](./WyPy.html){.nonexistent}\'s big brother, by the same author. (2004)

- [WyPy](http://c2.com/cgi/wiki?WyPy "Wiki"){.interwiki} \-- a [ShortestWikiContest](http://c2.com/cgi/wiki?ShortestWikiContest "Wiki"){.interwiki} contender, 23-, 18-, and 11-line versions. (2004)

## Archaeology {#Archaeology}

The following projects are not available anymore:

- [CyberPublishing](http://c2.com/cgi/wiki?CyberPublishing "Wiki"){.interwiki} \-- a wiki-based web publishing framework written in Python; GPL

- [Hatta](http://hatta-wiki.org){.http} \-- a Wiki which runs out of a Mercurial repository whose pages are just the files in that repository.

- [mojibake](http://frotz.net/sakana/space/mojibake){.http} \-- based on [Sakana](http://frotz.net/sakana/space/home){.http}, runs with [ModPython](ModPython)

- [MonkeyWiki](http://c2.com/cgi/wiki?MonkeyWiki "Wiki"){.interwiki} \-- another [PikiPiki](http://c2.com/cgi/wiki?PikiPiki "Wiki"){.interwiki} fork that claims to be simple to install/use, single file, very flexible templating, rewrite-friendly.

- [Not-Yet-Named Wiki](http://wiki.webwareforpython.org/thiswiki){.http} \-- runs on Webware. Supports static publishing, WYSIWYG editing, [reStructuredText](reStructuredText)

- [PikiePikie](http://moinmo.in/PikiePikie){.http} \-- Python-based, lets you put a [WebLog](http://c2.com/cgi/wiki?WebLog "Wiki"){.interwiki} on any page (based on [PikiPiki](http://c2.com/cgi/wiki?PikiPiki "Wiki"){.interwiki})

- [PikiPiki](http://c2.com/cgi/wiki?PikiPiki "Wiki"){.interwiki} \-- Python-based, completely standalone, GPL, famous ancestor of most Python wikies

- [Smug](http://www.mcnabbs.org/andrew/smug/){.http} - GPLed content editing system with moderation queue.

- [Sycamore](http://projectsycamore.org){.http} \-- seems to have roots in [MoinMoin](MoinMoin), but looks a little slicker while not as full featured. Uses a PostgreSQL or MySQL backend. GPL.

- [ZwiKi](http://c2.com/cgi/wiki?ZwiKi "Wiki"){.interwiki} \-- based in [ZoPe](http://c2.com/cgi/wiki?ZoPe "Wiki"){.interwiki}, powerful, GPL.
