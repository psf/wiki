# PyPiDocumentationHosting

::: {#content dir="ltr" lang="en"}
[http://pythonhosted.org/](http://pythonhosted.org/){.http} (formerly [http://packages.python.org/](http://packages.python.org/){.http} ) used to provide hosting for documentation for Python packages; this [is no longer the case](https://github.com/pypa/warehouse/issues/509){.https} and Python project developers should instead use [Read the Docs](https://readthedocs.org/){.https}.
:::
