# DeadLinks

::: {#content dir="ltr" lang="en"}
Python is often used to crawl the internet. One of the useful application for that is finding dead links.

## Theory {#Theory}

The basic method to check if link is dead is sending a HEAD request.

Advanced stuff requires crawling links that are alive, and skipping those that are already visited.

## Links to check {#Links_to_check}

- [PCrawler](http://math.nist.gov/~RPozo/ngraph/webcrawler.html){.http} - NIST modular crawler, Public Domain, needs some love.

- [weblinkchecker.py](https://www.mediawiki.org/wiki/Manual:Pywikibot/weblinkchecker.py){.https} - Wikipedia\'s Pywikibot link checker, MIT license.

- [https://pypi.python.org/pypi/LinkChecker](https://pypi.python.org/pypi/LinkChecker){.https} - many features, GPL.
