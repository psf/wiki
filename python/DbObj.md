# DbObj

::: {#content dir="ltr" lang="en"}
Database Objects, or dbObj for short, is a dynamic object-oriented, repository based layer on top of the Python DB-API 2.0 spec (PEP [0249](http://www.python.org/dev/peps/pep-0249 "PEP"){.interwiki}).

See [http://www.xs4all.nl/\~bsarempt/python/dbobj.html](http://www.xs4all.nl/~bsarempt/python/dbobj.html){.http}
:::
