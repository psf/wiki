[EpyDoc](http://epydoc.sourceforge.net/) is a documentation generator for Python similar to [JavaDoc](./JavaDoc.html). It provides its own simple text-markup language, but also supports the use of [reStructuredText](../archive/reStructuredText). It does not support Zope \"scarecrow\"-style interfaces.

Development of [EpyDoc](http://epydoc.sourceforge.net/) is hosted on [SourceForge](SourceForge).
