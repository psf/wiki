## Introduction 

The purpose of AnyGui project was (**development stopped in 2002**) to create a generic access API to different types of GUI libraries found to work with Python. The name is inspired by the standard Python module anydbm, which provides generic access to DBM-style databases.

## Supported platforms 

Current status of platform support [available](http://anygui.sourceforge.net/platforms.php) on the site. For the date of writing (16 September 2002) platform list was:

- [TkInter](TkInter)

- [BeThon](BeThon)

- Java Swing

- [PythonWin](../archive/PythonWin) (MSW via [Win32All](../platforms/Win32All))

- [PyGtk](PyGtk)

- [WxPython](../gui/WxPython)

- [PyQt](../gui/PyQt)

- Plain Text (also Curses)

Future plans are:

- Dynamic HTML

- PyFLTK

- PyUI

- PicoGUI ([http://picogui.org](http://picogui.org))
