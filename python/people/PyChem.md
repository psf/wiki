# PyChem

A graphical univariate & multivariate analysis package for WinXP and Linux. Features principal components analysis (PCA), discriminant function analysis (DFA), cluster analysis, PLS-DA & PLSR, genetic algorithms for feature selection.

See the project [webpage](http://pychem.sf.net) for further details, and our [OUP Bioinformatics](http://bioinformatics.oxfordjournals.org/cgi/content/abstract/22/20/2565) article.
