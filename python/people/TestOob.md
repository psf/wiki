# Testoob - Python Testing Out Of (The) Box 

## About 

[Testoob](http://testoob.sourceforge.net) is an advanced unit testing framework for Python. It integrates effortlessly with existing [PyUnit](PyUnit) (module \'unittest\') test suites.

[Version 1.1](http://sourceforge.net/project/showfiles.php?group_id=138557) was released on 24 June, 2006.

## Features 

Some of Testoob\'s features are:

- Work with [PyUnit](PyUnit) tests as-is, no changes necessary

- Filter which tests to run with regular expressions

- Output test results as XML/HTML/PDF

- Test skipping

- Color output on a terminal

- Run pdb post-mortem on failed tests

- Profiling support

- Run tests in parallel in threads or processes

- and more!
