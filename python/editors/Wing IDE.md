[Wing IDE](http://wingware.com/) is a powerful Python IDE with advanced debugger, editor with vi, emacs, visual studio, and other key bindings, auto-completion, auto-editing, goto-definition, find uses, refactoring, unit testing, code browser, multi-file search, and much more. Wing can debug code launched from the IDE, or remotely to another machine. Wing supports code running under Django, Zope, Plone, wxPython, [PyQt](PyQt), Blender, Maya, NUKE, and many other third party libraries and applications. The product comes in several levels, including free and paid options (\*).

See [wingware.com](http://wingware.com/) for details.

(\*) Free trial available with up to three 10 day trial periods.
