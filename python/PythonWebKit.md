# PythonWebKit

::: {#content dir="ltr" lang="en"}
[PythonWebKit](http://www.gnu.org/software/pythonwebkit){.http} provides direct python DOM bindings to the Webkit Web Browser engine. It places python as a peer of javascript, allowing developers to call functions such as getElementsByTagName, appendChild and even add python event callbacks onto XMLHttpRequest, Window and HTML Elements (onclick, onreadystatechange, onresize etc.)

The difference between PythonWebKit and [PyWebkitGtk](PyWebkitGtk) and [PyWebkitQt4](./PyWebkitQt4.html){.nonexistent} is that PythonWebKit adds the entire range of W3C DOM bindings functions - including HTML5 features - whereas [PyWebkitGtk](PyWebkitGtk) and [PyWebkitQt4](./PyWebkitQt4.html){.nonexistent} provide very little, inadequate, incomplete and more CPU-intensive DOM bindings.
:::
