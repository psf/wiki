# PyGeo

::: {#content dir="ltr" lang="en"}
PyGeo is a Dynamic 3d geometry laboratory.

See:

[http://pygeo.sourceforge.net](http://pygeo.sourceforge.net){.http}

It is pure Python code harnessing some of the power of

Numerical Python

[http://pfdubois.com/numpy/](http://pfdubois.com/numpy/){.http}

and

VPython

[http://www.vpython.org/](http://www.vpython.org/){.http}
