# PyQt/Third_Party_Packages_and_Modules

::: {#content dir="ltr" lang="en"}
This page contains a list of packages and modules that are designed to be used in PyQt and PyKDE applications. Some of these are also built with PyQt and PyKDE.

For a list of applications built with PyQt and PyKDE, see the [Some Existing Applications](./PyQt(2f)SomeExistingApplications.html) page.

## Camelot {#Camelot}

Camelot is a python GUI framework on top of Sqlalchemy and PyQt, inspired by the Django admin interface.\
Author: Conceptive Engineering\
Link: [http://www.conceptive.be/projects/camelot](http://www.conceptive.be/projects/camelot){.http}

## DaVinci Rendering Engine {#DaVinci_Rendering_Engine}

DaVinci is a vector graphics based declarative GUI toolkit for Python, which is currently under development.\
Author: K.S.Sreeram\
Link: [http://tachyon.in/davinci/](http://tachyon.in/davinci/){.http}

## fman build system {#fman_build_system}

fbs compiles the Python code of your PyQt application into a standalone executable and installer. It is based on PyInstaller but solves many of its common pain points. fbs open sources solutions that were originally implemented in months of labour for the cross-platform file manager [https://fman.io/](https://fman.io/){.https}.\
Link: [https://build-system.fman.io](https://build-system.fman.io){.https}

## IVuPy {#IVuPy}

IVuPy (I-View-Py) aims to be a solid basis for large Qt based Python programs geared to 3D visualization and data analysis of huge data sets.\
Author: Gerard Vermeulen\
Link: [http://ivupy.sourceforge.net/](http://ivupy.sourceforge.net/){.http}

## Matplotlib (Qt backend) {#Matplotlib_.28Qt_backend.29}

matplotlib is a python 2D plotting library which produces publication quality figures in a variety of hardcopy formats and interactive environments across platforms.\
Author: John Hunter and contributors\
Link: [http://matplotlib.sourceforge.net/](http://matplotlib.sourceforge.net/){.http}

## OpenAlea {#OpenAlea}

OpenAlea is an open source project primarily aimed at the plant research community, with a particular focus on Plant Architecture Modeling at different scales.\
Author: OpenAlea Project contributors\
Link: [http://openalea.gforge.inria.fr/](http://openalea.gforge.inria.fr/){.http}

## Pâté {#P.2BAOI-t.2BAOk-}

Pâté (or just Pate) is a plugin for Kate on KDE3 that allows you to create new plugins in Python for Kate.\
Author: Paul Giannaros\
Link: [http://paul.giannaros.org/pate/](http://paul.giannaros.org/pate/){.http}

## Pivy {#Pivy}

Pivy is a Coin binding for Python. Coin is a high-level 3D graphics library with a C++ Application Programming Interface.\
Author: Tamer Fahmy, Systems in Motion (SIM)\
Link: [http://pivy.coin3d.org/](http://pivy.coin3d.org/){.http}

## PyGUIUnit {#PyGUIUnit}

PyGUIUnit is a framework created for testing GUIs.\
Authors: Fabio Zadrozny and Bruno da Silva de Oliveira\
Link: [http://pyguiunit.sourceforge.net/](http://pyguiunit.sourceforge.net/){.http}

## PyKHTML {#PyKHTML}

A Python module for writing website scrapers/spiders.\
Author: Paul Giannaros\
Link: [http://paul.giannaros.org/pykhtml/](http://paul.giannaros.org/pykhtml/){.http}

## PyPoppler {#PyPoppler}

[PyQt4](PyQt4)-based bindings to the Poppler PDF library.\
Author: Rajeev J. Sebastian\
Link: [http://svn.pardus.org.tr/uludag/trunk/playground/pypoppler-qt4/](http://svn.pardus.org.tr/uludag/trunk/playground/pypoppler-qt4/){.http}

## PyQGIS {#PyQGIS}

Bindings for libraries of [Quantum GIS](http://qgis.org/){.http} that enable rapid development of GIS applications.\
Author: Martin Dobias\
Link: [http://www.qgis.org/wiki/Python_Bindings](http://www.qgis.org/wiki/Python_Bindings){.http}

## PyQtGraph {#PyQtGraph}

PyQtGraph is a pure-python graphics library built on PyQt4 and numpy. It is intended for use in mathematics / scientific / engineering applications and was initially developed for the acq4 project.\
Maintainer: lcampagn\
Link: [https://launchpad.net/pyqtgraph](https://launchpad.net/pyqtgraph){.https}

## PyQwt {#PyQwt}

PyQwt is a set of Python bindings for the Qwt C++ class library which extends the Qt framework with widgets for scientific and engineering applications.\
Author: Gerard Vermeulen\
Link: [http://pyqwt.sourceforge.net/](http://pyqwt.sourceforge.net/){.http}

## Pywidget KDM Patch {#Pywidget_KDM_Patch}

This patch makes it possible to embed widgets developed with [PyQt](PyQt) in a KDM theme.\
Author: Dental On Line\
Link: [http://opensource.dental-on-line.com/?page_id=18](http://opensource.dental-on-line.com/?page_id=18){.http}

## PyQt3Support {#PyQt3Support}

Independently maintained Qt3Support wrappers for [PyQt4](PyQt4).\
Author: Matteo Bertini \<[pyqt3support@develer.com](mailto:pyqt3support@develer.com){.mailto}\>\
Link: [http://www.develer.com/oss/PyQt3Support](http://www.develer.com/oss/PyQt3Support){.http}

## Q3 {#Q3}

(dead/unmaintained) Another independently maintained Qt3Support wrappers for [PyQt4](PyQt4).\
Author: James Lamanna\
Link: [http://kscraft.svn.sourceforge.net/viewvc/kscraft/Q3/trunk/](http://kscraft.svn.sourceforge.net/viewvc/kscraft/Q3/trunk/){.http}

## qcmwidgets

QcmWidgets are a collection of controls (widgets) for the Qt Ui Framework. The widgets are implemented in Python using [PyQt](PyQt).\
Author: chris.pm\
Link: [http://code.google.com/p/qcmwidgets/](http://code.google.com/p/qcmwidgets/){.http}

## SVGPaintEngine {#SVGPaintEngine}

(Part of the Veusz project - see the entry on the [SomeExistingApplications](./PyQt(2f)SomeExistingApplications.html) page.)\
Author: Jeremy Sanders\
Link: [http://svn.gna.org/viewcvs/veusz/trunk/document/svg_export.py?rev=1127&view=markup](http://svn.gna.org/viewcvs/veusz/trunk/document/svg_export.py?rev=1127&view=markup){.http}
