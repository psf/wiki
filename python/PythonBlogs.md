# PythonBlogs

::: {#content dir="ltr" lang="en"}
For collected weblogs related to Python, see [Planet Python](http://www.planetpython.org/){.http}.

Here also is an aggregator to keep track of python-related news, weblogs, wikis, and mailing lists: [http://www.bloglines.com/public/python](http://www.bloglines.com/public/python){.http}

If you have any suggested additions, post them here and I can add them to the aggregator.

You can import these subscriptions into your own aggregator using this link: [http://www.bloglines.com/export?username=python](http://www.bloglines.com/export?username=python){.http}
