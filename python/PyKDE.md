# PyKDE {#PyKDE-1}

PyKDE, written and actively maintained by Jim Bublitz, is a full interface to KDE\'s libraries. This allows you to write fully-fledged KDE applications with python.

PyKDE needs [PyQt](PyQt) and [SIP](SIP) to run. These are both actively maintained by Phil Thompson.

[Main Website](http://sourceforge.net/projects/pykde/){.http}

Documentation is available at [Riverbank Computing Limited](http://riverbankcomputing.co.uk){.http}.

If you have written an application for PyKDE, please list it here:

- [Python KIOSlave Tutorial](http://www.boddie.org.uk/david/Projects/Python/KDE/Docs/kioslaves-tutorial.html){.http}

- The [PyQt](PyQt) Wiki contains a [list of applications using PyQt and/or PyKDE](https://wiki.python.org/moin/PyQt/SomeExistingApplications){.https}.

- [Frescobaldi](http://www.frescobaldi.org/){.http} is a LilyPond music text editor written in Python using PyKDE4

- [pyRad](http://bitbucket.org/ArneBab/pyrad/){.http} is a wheel type command interface to allow quick access to programs.
