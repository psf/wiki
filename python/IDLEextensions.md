# IDLEextensions

::: {#content dir="ltr" lang="en"}
This page is a collection point for people to post links to IDLE extension modules that they have written themselves.

[IdleX](http://idlex.sourceforge.net/extensions.html){.http} provides a large collection of extensions to improve IDLE.

Alias: enhances IDLE\'s command-line with an alias facility and more. Download @ [lwickjr](lwickjr).

Edit: an edit command for Alias that attempts to invoke the IDLE editor on the source code of the object referenced on the command line. Download @ [lwickjr](lwickjr).

**Cheeseshop / Pypi** has a listing of IDLE extensions at [Framework :: IDLE](https://pypi.python.org/pypi?:action=browse&amp;c=520){.https}
:::
