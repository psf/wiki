\# \# Aren\'t URLs tedius? Have you ever wished you could just say the \# name of a web page, and get right to it? Here\'s something you can \# use to jump to any page in the Python Library Reference, by name, \# or any page of this wiki. \# \# \# Make a special bookmark in Firefox to do this: \# \# 1. Under \"Bookmarks,\" pick \"Manage Bookmarks\" \# \# 2. Create a new bookmark, call it \"Python Reference,\" \# and set the Location to: \# \# \"[http://localnames.taoriver.net/pythonref/%s](http://localnames.taoriver.net/pythonref/%s)\" \# \# Set the keyword to \"python\". \# \# 3. Put the bookmark in the \"Quick Searches\" folder. \# \# \# Now, at the address bar, you can just type something like: \# \# python cgi \# \# \...and you\'ll be at the Python Reference CGI page. \# \# Or type something like: \# \# python [FrontPage](FrontPage) \# \# \...and you\'ll be at the front page of the wiki. \# \# Take care! \# \# \# (If you\'re curious how this works, it\'s all just a bunch \# of webservices and browser redirection.) \# \# ( You can pull down the namespace binding with the following \# code, if you like, as well: \# \# import xmlrpclib \# nameserver = xmlrpclib.[ServerProxy](./ServerProxy.html)( \"[http://services.taoriver.net:9089/](http://services.taoriver.net:9089/)\" ) \# nameserver.parse_description( \"[http://wrap.localnames.taoriver.net/spaces/pythoninfo/](http://wrap.localnames.taoriver.net/spaces/pythoninfo/)\" ) \# \# You\'ll get back a big fat dictionary of name-URL bindings. \# See [http://services.taoriver.net:9089/](http://services.taoriver.net:9089/) for other functionality available to you. ) \#
