## Python Community Server 

The Python Community Server is a reimplementation of the [Radio Community Server](http://rcs.userland.com/) in Python. It can be used to host weblogs that are built with [Radio Userland](http://radio.userland.com), [PyDS](PyDS) or [bzero](http://www.myelin.co.nz/bzero/) or any other client that implements the XML storage system.

It is a Sourceforge project at [http://pycs.sourceforge.net/](http://pycs.sourceforge.net/)
