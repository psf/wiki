**Here is a list of known Python-based [WikiEngines](http://c2.com/cgi/wiki?WikiEngines "Wiki") sorted by status and alphabetical order.**

## Active Wikis 

- [MoinMoin](MoinMoin) \-- an evolution over [PikiPiki](http://c2.com/cgi/wiki?PikiPiki "Wiki") that you\'re using here - multiple ways to run, plugin-based architecture with lots of plugins and large community, GPL2. (2022)

- [Pwic.wiki](https://pwic.wiki) \-- extendable wiki integrated with Markdown, SQLite and [OpenDocument](./OpenDocument.html) Text, AGPL3. (2022)

- [SahrisWiki](http://sahriswiki.org) \-- Mercurial backend support, plugins, MIT licensed. (2016)

- [Sphene Community Tools](http://sct.sphene.net) \-- Django based wiki and forum applications. (2018)

- [Trac](http://trac.edgewall.org) \-- an enhanced wiki, issue tracking and source code browser system. Supports multiple markup formats including [MoinMoin](MoinMoin), [reStructuredText](reStructuredText), html, and [textile](http://www.textism.com/tools/textile/). Runs as [FastCGI](http://fastcgi.com), WSGI or standalone. Pluggable architecture. Modified BSD License. (2022)

- [zim-wiki](http://www.zim-wiki.org) - WYSIWIG personal/desktop wiki with index, links, image inclusion, checkboxes and bulleted lists, equations, graphviz; calendar, [ToDo](./ToDo.html); prints to clean html. Customizable, all-python over gtk2, inbuilt-server, uses filesystem as db; debian packaged. Documentation. GPL2. (2021)

## Unsupported Wikis 

The following projects are available but no more maintained:

- [Luminotes](https://github.com/osborne6/luminotes) \-- a WYSIWYG personal wiki notebook for organizing your notes and ideas. (2014)

- [Markdoc](https://github.com/zacharyvoase/markdoc) \-- a lightweight Markdown-based Wiki system, public domain. (Claimed as inactive)

- [Pyle](https://leastfixedpoint.com/tonyg/kcbbs/projects/pyle.html) \-- GPL wiki with multiple VCS backends, email notifications, edit by FTP and lot of other features. (2012)

- [Pwyky](http://infomesh.net/pwyky/) \-- [WyPy](./WyPy.html)\'s big brother, by the same author. (2004)

- [WyPy](http://c2.com/cgi/wiki?WyPy "Wiki") \-- a [ShortestWikiContest](http://c2.com/cgi/wiki?ShortestWikiContest "Wiki") contender, 23-, 18-, and 11-line versions. (2004)

## Archaeology 

The following projects are not available anymore:

- [CyberPublishing](http://c2.com/cgi/wiki?CyberPublishing "Wiki") \-- a wiki-based web publishing framework written in Python; GPL

- [Hatta](http://hatta-wiki.org) \-- a Wiki which runs out of a Mercurial repository whose pages are just the files in that repository.

- [mojibake](http://frotz.net/sakana/space/mojibake) \-- based on [Sakana](http://frotz.net/sakana/space/home), runs with [ModPython](ModPython)

- [MonkeyWiki](http://c2.com/cgi/wiki?MonkeyWiki "Wiki") \-- another [PikiPiki](http://c2.com/cgi/wiki?PikiPiki "Wiki") fork that claims to be simple to install/use, single file, very flexible templating, rewrite-friendly.

- [Not-Yet-Named Wiki](http://wiki.webwareforpython.org/thiswiki) \-- runs on Webware. Supports static publishing, WYSIWYG editing, [reStructuredText](reStructuredText)

- [PikiePikie](http://moinmo.in/PikiePikie) \-- Python-based, lets you put a [WebLog](http://c2.com/cgi/wiki?WebLog "Wiki") on any page (based on [PikiPiki](http://c2.com/cgi/wiki?PikiPiki "Wiki"))

- [PikiPiki](http://c2.com/cgi/wiki?PikiPiki "Wiki") \-- Python-based, completely standalone, GPL, famous ancestor of most Python wikies

- [Smug](http://www.mcnabbs.org/andrew/smug/) - GPLed content editing system with moderation queue.

- [Sycamore](http://projectsycamore.org) \-- seems to have roots in [MoinMoin](MoinMoin), but looks a little slicker while not as full featured. Uses a PostgreSQL or MySQL backend. GPL.

- [ZwiKi](http://c2.com/cgi/wiki?ZwiKi "Wiki") \-- based in [ZoPe](http://c2.com/cgi/wiki?ZoPe "Wiki"), powerful, GPL.
