[ActiveState](http://www.ActiveState.com/) offers support through [ActivePython Business Edition](http://www.activestate.com/business-edition) (per-server licensing) and [ActivePython Enterprise Edition](http://www.activestate.com/enterprise-edition). Support can also be included as part of [ActivePython OEM re-distribution agreements](http://www.activestate.com/oem).

[eGenix.com](http://www.egenix.com/) offers Python, Django, Plone and Zope support, as well as product support for our wide range of related [software products](http://www.egenix.com/products/), including our own [eGenix PyRun](http://www.egenix.com/products/python/PyRun/) Python runtime distribution which focuses simplifying Python software product distribution.

[Enthought](http://www.enthought.com/) offers support, documentation, and commercial licensing for the [Enthought Python Distribution](http://www.enthought.com/products/epd.php/). EPD provides a Python environment for scientists, engineers, and data analysts.

[Logilab](http://www.logilab.com/) offers support contract for companies developping with Python (and doing scientific stuff)

[tummy.com](http://www.tummy.com/) offers support for Python on Linux, including RPM packaging, hosting, and script development.
