Just van Rossum is a Dutch (typo)graphic designer / [MacPython](MacPython) guru. Started the [/LettError](./JustVanRossum(2f)LettError.html) virtual type foundry with [ErikVanBlokland](ErikVanBlokland), with whom he worked at [Meta Design](http://www.metadesign.com/). Fame came with the release of Beowolf (co-designed with [ErikVanBlokland](ErikVanBlokland)), a font whose ragged edges shift randomly each time you print the font. He\'s the developer of the [/MacPythonIDE](./JustVanRossum(2f)MacPythonIDE.html), designed the popular [Python Powered](http://starship.python.net/~just/pythonpowered/) logo, developed [/DrawBot](./JustVanRossum(2f)DrawBot.html) and [FontTools](./FontTools.html), helps out with [PyObjC](./PyObjC.html) and [MacPython](MacPython), etc.

We\'ve started a page with notes about learning Cocoa using [/PyObjC](./JustVanRossum(2f)PyObjC.html): LtrWiki:StartCocoa
