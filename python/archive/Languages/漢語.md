## BOOK 

- [Dive Into Python](http://diveintopython.org/cn/), a free Python tutorial for experienced programmers, has been translated into Chinese by limodou.

Read it online or [download it](http://diveintopython.org/download/cn/diveintopython.zip).

- Wesley Chun\'s **Core Python Programming** is now [available in Chinese](http://www.china-pub.com/computers/common/info.asp?id=3097). (China Machine Press, August 2001, 518 pp.)

- [简明 Python 教程（\"A Byte of Python\"）](http://www.ByteOfPython.info/language/chinese/index.html)／[繁體中文版](http://code.Google.com/p/zhpy/wiki/ByteOfZhpy)／[英文版官網](http://www.ByteOfPython.info)

- [電腦做什麼事 \-- Python，讓寫程式更簡單](http://pydoing.blogspot.com/)，Python的教學網誌，提供給完全初學者的系列文章，包含Python基礎，Pygame、wxPython及Django。
