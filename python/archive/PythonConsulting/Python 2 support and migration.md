A list of vendors, in alphabetical order, who provide services supporting Python 2 codebases or migrating from Python 2 to Python 3.

**Disclaimer:** *This page, like the rest of the Wiki, is user-contributed content; being listed here does not imply any sort of endorsement by the Python Software Foundation.*

- [Abilian (France)](https://abilian.com/fr/services/migration/)

- [ActiveState](https://www.activestate.com/products/activepython/python-2-7/):(Global) providing Python 2 to 3 migration assistance and long-term Python 2 support.

- [AXANT](http://python.axant.it/) (Italy)

- [Hendrikx ITC](https://www.hendrikx-itc.nl/) (The Netherlands) providing Python 2\>3 migration

- [iTechArt](https://www.itechart.com/blog/key-ways-to-switch-to-python-3/) *(USA, UK, Western Europe, and Eastern Europe)* **providing 2 to 3 migration** (not long term Python 2 support)

- [Level 12](https://www.level12.io/blog/why-we-use-python-in-a-single-tweet/): providing 2 -\> 3 migration, but not long term Python 2 support

- [Python Academy](https://www.python-academy.com/consulting/python_migration_2_to_3.html) (Germany)

- [Reckon Digital](https://reckondigital.com/) (United Kingdom)

- [REVSYS](https://www.revsys.com/services/python/)

- [Six Feet Up](https://sixfeetup.com/)

- [Snakedev](https://snake.dev/)

Some of the consulants listed at [PythonConsulting](PythonConsulting) may also be able to help, but if they\'re not listed here you\'ll have to ask them.
