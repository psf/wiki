# Electronic Design Automation Tools 

**and any other Python packages that might be helpful to circuit design engineers**

- [MyHDL](./MyHDL.html) - A Python-Based Hardware Description Language [http://www.myhdl.org/](http://www.myhdl.org/)

- [SimPy](SimPy) - A General Purpose Discrete Event Simulator [http://sourceforge.net/p/simpy/](http://sourceforge.net/p/simpy/)

- [PyCell Studio](./PyCell(20)Studio.html) - A stand-alone solution for [OpenAccess](./OpenAccess.html) PCell design. Free download. [http://www.ciranova.com](http://www.ciranova.com)

- [SourceForge Projects](http://sourceforge.net/directory/science-engineering/eda/) - Listed under Electronic Design Automation

- [gEDA](http://www.geda.seul.org/) - a full GPL\'d suite of EDA tools

- [EDA Tools Lab](http://ece.arizona.edu/~edatools) - Current focus is on use of Python in circuit design

- [OpenEDA](http://openeda.si2.org/) - OpenAccess and other standards in the EDA industry

- [PyEDA](https://github.com/cjdrake/pyeda) - Python EDA, Boolean Algebra
