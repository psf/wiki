## Other Recordings 

- [From Marakana TechTalk](http://marakana.com/forums/python/python/)

  - Marakana Python instructor, Simeon Franklin, is going to show you how you can debug your Python applications using the Python debugger, pdb.

- [From Python Import Podcast](http://frompythonimportpodcast.com/)

  - From Python Import Podcast is a roundtable podcast from several guys in Ohio. Make sure you check out the two part special (Episodes 002 and 003) on [The Zen of Python](http://www.awaretek.com/zen.html)!

- [A Little Bit of Python](http://bitofpython.com)

  - Commentary from some of the leading figures in the Python community on the current issues of interest.

- [Django Dose](http://djangodose.com/)

  - Podcast about the Django web framework.

- [Python411](http://www.awaretek.com/python/) ([podcast RSS](http://www.awaretek.com/python/index.xml))

- [Jython Podcast](http://www.jythonpodcast.com)

- [Python and other screencasts from COMP249 course](http://www.ics.mq.edu.au/~cassidy/2007/06/08/screencasts-in-teaching-web-technology/) by Steve Cassidy at Macquarie University, Australia.

- [Python e-textbook with audio tutorials](https://www.speltechnologies.com/bookdemo) Merscythe: Adventures with the Codue is a story-based game paired with audio tutorials for learning Python programming. Designed for accessibility.

- [Radio Free Python](http://radiofreepython.com/) - a monthly podcast featuring news and interviews

- [Zopecast](http://plope.com/zopecast)

  - Zopecast is a show recorded every so often by Chris [McDonough](./McDonough.html) and Tres Seaver with the aim of jawing about things in the Zope world.
