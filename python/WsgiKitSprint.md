# WsgiKitSprint

::: {#content dir="ltr" lang="en"}
This will be a sprint on [WSGIKit](http://svn.colorstudy.com/trunk/WSGIKit){.http}, and by association [Webware](http://webwareforpython.org){.http} and [WSGI](http://www.python.org/peps/pep-0333.html){.http}. Really the emphasis will be on building reusable framework-neutral WSGI middleware to help replicate (and improve!) Webware\'s functionality; so you don\'t have to be interested in Webware to participate, you just have to be interested in Python web programming. (I\'ll be writing some more information about the design of WSGIKit before the sprint, and moving it to a more public repostory \-- Ian)

Discussion should take place on [webware-devel@lists.sourceforge.net](mailto:webware-devel@lists.sourceforge.net){.mailto}
