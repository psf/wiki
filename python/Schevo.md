## Synopsis {#Synopsis}

Next-generation Object-Oriented DBMS

## Features {#Features}

- Rapid Development

- Rich Schema Definition

- Automated Schema Evolution

- Transaction Based

- User Interface Generation

- PersistentMapping Transactions

- BTrees

- Durus Backend Support

- ZODB Backend Support

- PyPy and Python 3 Support

## Supported Python applications {#Supported_Python_applications}

- [Django](Django)

- Extended Durus 3.9 and [ZODB](ZODB) support

- PyGTK (require python 2.7)

- gevent and PyPy

- uWSGI

## Pros {#Pros}

- Persistent storage on top of [Durus](Durus) and [ZODB](ZODB).

- epoll support

## Cons {#Cons}

- GTK2 bindings depends on Python 2.7
