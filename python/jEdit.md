jEdit is a nice Swing based editor for many languages and Java specially. You can add additional plugins like [JPyDbg](http://jpydbg.sourceforge.net/){.http} and Jython for your Python work.

One of the nicest feature of jEdit is the folding. Once you get used to it. It is very difficult to work without it. By the way, since version 6.0 [ViImproved (Vim)](./ViImproved(2028)Vim(29).html){.nonexistent} also has folding.

If you want a simpler, lighter and faster version, you might want to check out [JeXt](JeXt) or [J](http://armedbear-j.sf.net/){.http} I recently came across *J* and find it quite nice for doing Python.
