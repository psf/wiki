### Internationalization and localization efforts for Chinese, Japanese and Korean environments on Python. {#Internationalization_and_localization_efforts_for_Chinese.2C_Japanese_and_Korean_environments_on_Python.}

**CJKCodecs has been integrated into Python 2.4 and you will have it by default.**

- CJKCodecs - The CJKCodecs is a unified unicode codec set for Chinese, Japanese and Korean encodings. It supports full features of unicode codec specification and PEP293 error callbacks on Python 2.3. [http://cjkpython.berlios.de/](http://cjkpython.berlios.de/){.http}
