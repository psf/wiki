[1SEP09 Ideally, all the language pages should be like the Polish or Turkish pages - all native language, only the necessary English.]{.comment style="display:none"}

[There are some groundrules, some laid down by the site admins, some my suggestions:]{.comment style="display:none"}

[1) Pages must be named in ASCII and English ([PolishLanguage](PolishLanguage))]{.comment style="display:none"}

[2) Pages must have an explanation in English at the top (Links to Python information in \<language X\>)]{.comment style="display:none"}

[3) (my suggestion) We probably want to limit invites to edit the pages to people we know well, or Pythonistas with a track record. Hopefully this is inclusive enough without opening the site up to a spam flood and vandalismfest.]{.comment style="display:none"}

[Where these pages really need help:]{.comment style="display:none"}

[1) check links, remove broken ones.]{.comment style="display:none"}

[2) add new links that are quality Python information and active.]{.comment style="display:none"}

[3) some care for languages that have next to nothing, but do have people in the Python community - even a link to the Wikipedia page for Python, in that language, is a start (Some are pretty complete and of high quality - the Russian language Wikipedia page for Python, for instance, packs a lot in).]{.comment style="display:none"}

## Bücher {#B.2BAPw-cher}

- [Routineaufgaben mit Python automatisieren](https://dpunkt.de/produkt/routineaufgaben-mit-python-automatisieren-2/){.https} Die deutsche Übersetzung () des Buches \"Automate the Boring Stuff with Python\" von Al Schweigart in der 2. Auflage (jetzt für Python 3) ist eine Python-Einführung am Beispiel von alltäglichen Aufgaben eines Computernutzers. Al Schweigart stellt die [englische Originalausgabe](https://automatetheboringstuff.com/){.https} kostenlos zum online-Lesen zur Verfügung. [dpunkt.verlag](https://dpunkt.de/){.https}.

- [Python 3 Das umfassende Handbuch](http://openbook.rheinwerk-verlag.de/python/){.http} von Johannes Ernesti und Peter Kaiser, als Openbook aus dem [Rheinwerk Verlag](https://www.rheinwerk-verlag.de/){.https}. Alternativ auch als Buch/eBook gegen Bezahlung ![;-)](/wiki/europython/img/smile4.png ";-)"){height="16" width="16"}

- [Python von Kopf bis Fuß](https://oreilly.de/produkt/python-von-kopf-bis-fuss/){.https} von Paul Barry ist ein Buch für Einsteiger. [O\'Reilly Germany](http://www.oreilly.de/){.http}.

- [Programmieren lernen mit Python](https://oreilly.de/produkt/programmieren-lernen-mit-python/){.https} von Allen B. Downey ist an Einsteiger gerichtet. Die 2. Auflage behandelt Python 3.4.

- [A Byte of Python (deutsche Übersetzung)](http://cito.github.io/byte_of_python/){.http} ist die deutsche Übersetzung von Swaroop C Hs [A Byte of Python](http://www.swaroopch.com/notes/Python){.http} für Python 2.x.

- [Einstieg in Python](https://www.rheinwerk-verlag.de/einstieg-in-python-ideal-fuer-programmiereinsteiger/){.https}, von Thomas Theis, ist ein deutsches Buch für Anfänger. [Rheinwerk Verlag](https://www.rheinwerk-verlag.de/){.https}.

- [Python kurz & gut](https://oreilly.de/produkt/python-kurz-gut/){.https} ist eine Übersetzung von \"Python Pocket Reference\" (Mark Lutz) von Stefan Fröhlich und ist auf dem Stand von Python 2.7 und 3.4. [O\'Reilly Germany](http://www.oreilly.de/){.http}.

- [Programmieren lernen](http://www.alan-g.me.uk/tutor/german/index.htm){.http} ist eine Übersetzung (durch Bruno Schäfer) von Alan Gaulds [Learning to Program](http://www.alan-g.me.uk/l2p/){.http}.

- [Python für Kids](http://python4kids.net){.http} - Gregor Lingls Buch, das Kinder in der Mittel- und Oberstufe an Python heranführt. Die letzte, 4. Auflage benutzt Python 3.1.

- [Python & XML](http://www.oreilly.de/catalog/pythonxmlger/){.http} ist eine Übersetzung von \"Python and XML\" (Christopher A. Jones & Fred L. Drake, Jr.) von Dinu Gherman. [O\'Reilly Germany](http://www.oreilly.de/){.http}.

- [Einführung in Python 3](https://www.hanser-fachbuch.de/buch/Einfuehrung+in+Python+3/9783446452084){.https} ist ein Buch von [Bernd Klein](http://www.bklein.de){.http}, was besonders geeignet ist für Programmieranfänger, aber auch für Umsteiger von anderen Programmiersprachen. [Hanser Verlag](http://www.hanser-fachbuch.de){.http}.

- [Hitchhiker's Guide für Python](https://oreilly.de/produkt/hitchhikers-guide-fuer-python/){.https} Kenneth Reitz und Tanya Schlusser befasst sich mit Python Best Practices. [O\'Reilly Germany](http://www.oreilly.de/){.http}.
