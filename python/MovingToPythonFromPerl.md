# Moving to Python from Perl {#Moving_to_Python_from_Perl}

## Books {#Books}

There is a book on moving from Perl to Python:

- \"Perl to Python Migration\" by Martin C. Brown, ISBN [0201734885](http://www.amazon.com/exec/obidos/ISBN=0201734885 "ISBN"){.interwiki}, Addison Wesley, 320 pages (2002).

  \"Are you a Perl programmer who would like to be able to programme in Python? Brimming with a wealth of practical examples and resources this book will show you how.\" [More information](http://www.pearsonptg.com/book_detail/0,3771,0201734885,00.html){.http}.
