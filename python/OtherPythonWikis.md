# Thinki salvage {#Thinki_salvage}

Thinki [http://thinkware.se/cgi-bin/thinki.cgi/](http://thinkware.se/cgi-bin/thinki.cgi/){.http} used to be a second major python wiki, similar in scope to wiki.python.org.

It appears to be gone. The Internet Archive last [captured](http://web.archive.org/web/*/http://www.thinkware.se/cgi-bin/thinki.cgi/){.http} the homepage on [Dec 2007](http://web.archive.org/web/20071218211846/http://www.thinkware.se/cgi-bin/thinki.cgi/){.http}, spammed. The previous [Oct 2007](http://web.archive.org/web/20071011181006/http://thinkware.se/cgi-bin/thinki.cgi/){.http} homepage capture looks intact.
