## Other Recordings {#Other_Recordings}

- [From Marakana TechTalk](http://marakana.com/forums/python/python/){.http}

  - Marakana Python instructor, Simeon Franklin, is going to show you how you can debug your Python applications using the Python debugger, pdb.

- [From Python Import Podcast](http://frompythonimportpodcast.com/){.http}

  - From Python Import Podcast is a roundtable podcast from several guys in Ohio. Make sure you check out the two part special (Episodes 002 and 003) on [The Zen of Python](http://www.awaretek.com/zen.html){.http}!

- [A Little Bit of Python](http://bitofpython.com){.http}

  - Commentary from some of the leading figures in the Python community on the current issues of interest.

- [Django Dose](http://djangodose.com/){.http}

  - Podcast about the Django web framework.

- [Python411](http://www.awaretek.com/python/){.http} ([podcast RSS](http://www.awaretek.com/python/index.xml){.http})

- [Jython Podcast](http://www.jythonpodcast.com){.http}

- [Python and other screencasts from COMP249 course](http://www.ics.mq.edu.au/~cassidy/2007/06/08/screencasts-in-teaching-web-technology/){.http} by Steve Cassidy at Macquarie University, Australia.

- [Python e-textbook with audio tutorials](https://www.speltechnologies.com/bookdemo){.https} Merscythe: Adventures with the Codue is a story-based game paired with audio tutorials for learning Python programming. Designed for accessibility.

- [Radio Free Python](http://radiofreepython.com/){.http} - a monthly podcast featuring news and interviews

- [Zopecast](http://plope.com/zopecast){.http}

  - Zopecast is a show recorded every so often by Chris [McDonough](./McDonough.html){.nonexistent} and Tres Seaver with the aim of jawing about things in the Zope world.
