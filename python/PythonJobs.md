## International Job Boards {#International_Job_Boards}

- [Python Job Board](https://www.python.org/jobs/){.https} - maintained by volunteers of the PSF

- [OpenStack jobs](https://www.openstack.org/community/jobs/){.https} - all [OpenStack](./OpenStack.html){.nonexistent} is written in Python and most [OpenStack](./OpenStack.html){.nonexistent} companies are recruiting

- [r/PythonJobs](https://www.reddit.com/r/PythonJobs/){.https} - subreddit with regular Python job postings

- [pythonjobs.github.io](https://pythonjobs.github.io/){.https} - another volunteer run effort to build a Python jobs board (*Feb 2025: no longer receiving updates, it seems*)

## Regional Job Boards {#Regional_Job_Boards}

### Germany {#Germany}

- [Job board of the German Arbeitsagentur](https://www.arbeitsagentur.de/jobsuche/suche?angebotsart=1&was=Python){.https} - postings listed by the federal employment agency in Germany

Note: this page is a meta job board rather than a job board itself, so limit any links below to search links for corporate sites that regularly advertise positions for Python developers. Don\'t post links to individual positions.

- [Red Hat](https://redhat.wd5.myworkdayjobs.com/jobs/?q=python){.https} - Red Hat uses Python heavily for upstream projects, commercially supported products, professional consulting services and internal tools development.
