# Audio and Video 

Some Python-based projects related to audio and video. See also [Screencasting](Screencasting).

## Audio/Music Editors 

- [Jokosher](http://www.jokosher.org/) - a complete application for recording, editing, mixing and exporting audio

See also [PythonInMusic](PythonInMusic) for more projects.

## Audio/Music Players 

- [FUPlayer](http://fuplayer.org/) - a music player with management and CD ripping/burning capabilities

- [Listen](http://www.listen-project.org/) - an audio player with music management features

- [Quod Libet](http://www.sacredchao.net/quodlibet) - an audio player with scalable management of music collections (more than \"a mere 10000 songs\")

- [TimPlayer](TimPlayer) - a PyGTK based music player using GStreamer (see [http://creationix.com/OpenSource/TimPlayer/](http://creationix.com/OpenSource/TimPlayer/))

## Libraries and Frameworks 

- [GStreamer](http://www.gstreamer.net/) - an audio/video playback framework - has Python bindings

- [MLT](http://www.mltframework.org/) - an open source multimedia framework, designed and developed for television broadcasting - has Python bindings

- [MoviePy](http://zulko.github.io/moviepy/) - a Python library for script-based movie editing (cuts, concatenations, text insertion, non-linear editing)

See also [Audio](Audio) for details of Python audio libraries.

## Media Centres 

- [Elisa](http://elisa.fluendo.com/) - an open source cross-platform media centre solution

- [Freevo](http://freevo.sourceforge.net/) - Linux multimedia jukebox

## Streaming 

- [Boxtream](http://boxtream.unice.fr/) - a mobile and autonomous audio and video streaming and recording studio (uses GStreamer)

- [Flumotion](http://www.flumotion.net/) - a streaming media server (uses GStreamer)

## Video Editors 

- [OpenShot](http://www.openshotvideo.com/) - an open source video editor, written in Python and based on MLT and [PyGtk](PyGtk)

- [Pitivi](http://www.pitivi.org/) - an open source video editor, written in Python and based on GStreamer and GTK+
