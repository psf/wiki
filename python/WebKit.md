# 1. Webkit {#Webkit}

There are two meanings to \"webkit\". The major one is the web browser engine, called webkit - [http://www.webkit.org](http://www.webkit.org){.http} - see [WebBrowserProgramming](WebBrowserProgramming) for details. The second meaning is as a component of a server-side web development framework called Webware. see [Webware](./Webware.html) for details.
