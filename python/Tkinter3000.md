# Tkinter3000

::: {#content dir="ltr" lang="en"}
Tkinter3000 is a collection of Tkinter extensions, by [FredrikLundh](FredrikLundh).

The most polished part is the [WidgetConstructionKit](WidgetConstructionKit) (WCK), which allows you to write new Tkinter widgets in pure Python.

More info here: [http://tkinter.effbot.org](http://tkinter.effbot.org){.http}
:::
