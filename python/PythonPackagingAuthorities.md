# PythonPackagingAuthorities

::: {#content dir="ltr" lang="en"}
The Python Packaging Authority (PyPA) is a working group that maintains many of the relevant projects in Python packaging.

They maintain a site at [https://www.pypa.io](https://www.pypa.io){.https}, host projects on github and bitbucket, and discuss issues on the pypa-dev mailing list.
:::
