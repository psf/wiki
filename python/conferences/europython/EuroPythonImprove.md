# EuroPython Improvements 

This page is a collection of links to pages about improving the [EuroPython](EuroPython) conference, notably how it is organised as well as the general experience.

- [EuroPythonRoles](EuroPythonRoles) - organiser and participant roles in [EuroPython](EuroPython)

Please join the europython-improve mailing list [(info)](http://mail.python.org/mailman/listinfo/europython-improve) for discussion of [EuroPython](EuroPython) improvements.

If you wish to volunteer to help [EuroPython](EuroPython) in some way, consider joining the europython-volunteer-announce list [(info)](http://mail.python.org/mailman/listinfo/europython-volunteer-announce) to receive notifications on this topic.
