# StaticSiteGenerator

::: {#content dir="ltr" lang="en"}
Static site generator is a software that takes some **text + [templates](Templating)** as input and produces **html files** on the output. A picture:

       ┌──────┐   ┌───────────┐   ┌─────────────┐ **
       │ text │ + │ templates │ = │ .html files │
       └──────┘   └───────────┘   └─────────────┘

Generated site can be a blog, a [game list](https://osgameclones.com){.https}, [interactive demo](http://interactive.blockdiag.com){.http} or anything else.

## Static site generators written in Python {#Static_site_generators_written_in_Python}

- [VenC](https://github.com/DenisSalem/VenC){.https} - Markdown/Yaml static blog generator with multiple columns layout support. Aim to be light and easy to use. License is GNU/GPLv3

- [Clay](https://lucuma.github.io/Clay/){.https} - Jinja2 for easy site prototyping, MIT.

- [Complexity](https://complexity.readthedocs.io/){.https} - A refreshingly simple static site generator, for those who like to work in HTML.

- [Cyrax](https://github.com/piranha/cyrax){.https} - poetic Jinja2 engine, used for [osgameclones](https://osgameclones.com){.https}.

- [html2text](https://github.com/aaronsw/html2text){.https} - useful tool to convert HTML into Markdown.

- [Hyde](https://hyde.github.io){.https} - Jinja2-based static web site generator, MIT, [source](https://github.com/hyde/hyde){.https}

- [Mynt](https://mynt.uhnomoli.com){.https} - Static website generator based on Markdown and Jinja2. BSD-3.

- [Nikola](https://getnikola.com){.https} - Generator supporting reST, Markdown, IPyNB et al.; using Mako and Jinja2 for templates; supports multilingual sites, galleries, RSS feeds, DISQUS et al. for comments. Incremental builds. MIT license.

- [Obraz](https://obraz.pirx.ru){.https} - Static blog-aware site generator mostly compatible with Jekyll, MIT

- [Poole](https://hg.sr.ht/~obensonne/poole){.https} - static website generator, Markdown. GPL.

- [Pelican](https://github.com/getpelican/pelican/){.https} - Uses Markdown or ReST for content and Jinja 2 for themes. Supports DVCS, Disqus. AGPL.

- [PubTal](https://www.owlfish.com/software/PubTal/){.https} - [Templating#SimpleTAL](Templating#SimpleTAL)-based static site generator, BSD-3

- [rest2web](http://www.voidspace.org.uk/python/rest2web/){.http} - Generates Websites from ReST contents, BSD

- [Sphinx](https://www.sphinx-doc.org){.https} - Python\'s official documentation system that turns ReST into HTML, LaTeX, man pages, plaintext, and has many features for making site trees, inter-linking, and other documentation necessities (Jinja2 as default template engine). BSD.

- [staticninja](https://github.com/Ceasar/staticjinja){.https} - MIT, Jinja2, auto-compile. That simple.

- [tahchee](https://pypi.python.org/pypi/Tahchee/1.0.0){.https} - [Cheetah](Cheetah)-based static web site generator

- [tinkerer](https://github.com/vladris/tinkerer){.https} - blogging engine/static website generator powered by Sphinx with polite-correct license list.

- [Urubu](https://urubu.jandecaluwe.com){.https} - A micro CMS for static websites, with a focus on good navigation practices. AGPL.

These are outdated, but worth mentioning:

- [https://www.nthwave.net/elements](https://www.nthwave.net/elements){.https} - powerful tool for experienced web developers from the year 2002, requires at least Python 2.2

See also [static blog generators](https://wiki.python.org/moin/PythonBlogSoftware#Static){.https}.
