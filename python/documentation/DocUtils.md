[Docutils](http://docutils.sourceforge.net/) is an open-source text processing system for processing plaintext documentation into useful formats, such as HTML or LaTeX. It includes [reStructuredText](http://docutils.sourceforge.net/rst.html), the easy to read, easy to use, what-you-see-is-what-you-get plaintext markup language.
