# PythonWikipediabot

::: {#content dir="ltr" lang="en"}
The Python Wikipediabot Framework, also called pywikipedia is a set of bot scripts to do things on Wikipedia or other wikis using [MediaWiki](http:\\www.mediawiki.org){.http}. It has been written by several people using the language Python. It is available form [SourceForge](http://sourceforge.net/projects/pywikipediabot/){.http} or via SVN at [http://svn.wikimedia.org/svnroot/pywikipedia/trunk/pywikipedia/](http://svn.wikimedia.org/svnroot/pywikipedia/trunk/pywikipedia/){.http} For help go to [#pywikipediabot](irc://irc.freenode.net/pywikipediabot){.irc} @ freenode server or use the [pywikipediabot mailing list](http://lists.wikimedia.org/mailman/listinfo/Pywikipedia-l){.http}. A guide to pywikipedia can be found [here](http://meta.wikimedia.org/wiki/Using_the_python_wikipediabot){.http}. There is also a wiki where is possible to test your scripts, [Botwiki](http://botwiki.sno.cc/wiki/Main_Page){.http}.
:::
