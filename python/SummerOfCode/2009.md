# SummerOfCode/2009

:::: {#content dir="ltr" lang="en"}
This page coordinates the [Google Summer of Code](http://code.google.com/soc/){.http} projects involving Python and mentored by the Python Software Foundation (PSF) in 2009.

The 2009 PSF SoC coordinator is [ArcRiley](ArcRiley) \<[arcriley@gmail.com](mailto:arcriley@gmail.com){.mailto}\>. Contact him if you have any questions.

# Prospective Students {#Prospective_Students}

If you are a student interested in working on core Python development or on a project that helps the Python community, we\'d love to have you apply to the Python Software Foundation for this year\'s Google Summer of Code.

You should join the [soc2009-general](http://mail.python.org/mailman/listinfo/soc2009-general){.http} mailing list and take a look at the Project Ideas section below. If any interest you, feel free to contact the proposer for details. You can also discuss your own project ideas with the people mentioned or talk about them on the soc2009-general mailing list. Some of us also hang out on #gsoc-python on freenode.

Students should read [SummerOfCode/Expectations](./SummerOfCode(2f)Expectations.html) to understand what is expected of them.

# Project Ideas {#Project_Ideas}

These are project ideas proposed by mentors grouped roughly on the type of work it entails. Use the provided links for each for more information and contact the mentors interested in it.
