# GmPy

::: {#content dir="ltr" lang="en"}
The General Multiprecision PYthon project (GMPY) focuses on Python-usable modules providing multiprecision arithmetic functionality to Python programmers.

Some informational text and **old** download files are at [http://gmpy.sourceforge.net/](http://gmpy.sourceforge.net/){.http}

The project has now moved to google-code: [http://code.google.com/p/gmpy](http://code.google.com/p/gmpy){.http}

\...which has been moved to [GitHub](./GitHub.html){.nonexistent}: [https://github.com/aleaxit/gmpy](https://github.com/aleaxit/gmpy){.https}

*GNU Multiple Precision Arithmetic Library [http://gmplib.org](http://gmplib.org){.http}*
