# AnyGui

::: {#content dir="ltr" lang="en"}
## Introduction {#Introduction}

The purpose of AnyGui project was (**development stopped in 2002**) to create a generic access API to different types of GUI libraries found to work with Python. The name is inspired by the standard Python module anydbm, which provides generic access to DBM-style databases.

## Supported platforms {#Supported_platforms}

Current status of platform support [available](http://anygui.sourceforge.net/platforms.php){.http} on the site. For the date of writing (16 September 2002) platform list was:

- [TkInter](TkInter)

- [BeThon](BeThon)

- Java Swing

- [PythonWin](PythonWin) (MSW via [Win32All](Win32All))

- [PyGtk](PyGtk)

- [WxPython](WxPython)

- [PyQt](PyQt)

- Plain Text (also Curses)

Future plans are:

- Dynamic HTML

- PyFLTK

- PyUI

- PicoGUI ([http://picogui.org](http://picogui.org){.http})
