## Pro Python System Administration 

- Rytis Sileika

ISBN: [1430226056](http://www.amazon.com/exec/obidos/ISBN=1430226056 "ISBN") Apress, 416 pages (July 15, 2010)

This book explains and shows how to apply Python scripting in practice. In this book, you will find several projects in the categories of network administration, web server administration, and monitoring and database management.

More information: [http://www.sysadminpy.com](http://www.sysadminpy.com)

ISBN: [0596515820](http://www.amazon.com/exec/obidos/ISBN=0596515820 "ISBN"), O\'Reilly Media, 464 pages (August 22, 2008)
