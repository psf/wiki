# Overviews and Guides 

This page contains a list of overviews and guides that aim to provide more depth and background material than the items on the [Sample Code](./PyQt(2f)SampleCode.html) page.

- [Graphics View - Text Effects](./PyQt(2f)Graphics_View_(2d)_Text_Effects.html) - an example showing simple text effects using the [Graphics View framework](http://doc.trolltech.com/4.2/graphicsview.html).

- [pyqt4 and twisted](./PyQt(2f)pyqt4_and_twisted.html) - a (stub) tutorial on how to use [twisted](http://twistedmatrix.com) with [pyqt4](http://www.riverbankcomputing.co.uk/pyqt/download.php).

- [PyKDE and xattr](./PyQt(2f)PyKDE_and_xattr.html) - a tutorial and script on how to make a simple PyKDE application.

- [Multimedia Resources](./PyQt(2f)Multimedia_Resources.html) - the beginning of a document about using audio and video libraries with PyQt and PyKDE.

- [Using Python Custom Widgets in Qt Designer](./PyQt(2f)Using_Python_Custom_Widgets_in_Qt_Designer.html) - shows how to prepare an example widget so that it can be used in Qt Designer.

- [MathTran Equation Editor](./PyQt(2f)MathTran_Equation_Editor.html) - shows how to use `QHttp`{.backtick} to obtain data from a Web service.

- [Modifying a standard Qt image plugin](./PyQt(2f)Modifying_a_standard_Qt_image_plugin.html) - more about hacking Qt than creating beautiful things with Python

- [Designing Custom Controls with PyQt](http://doc.trolltech.com/qq/qq26-pyqtdesigner.html) - an article in Qt Quarterly 26 with downloadable example code.

  - See the article\'s [page on the Qt Centre Wiki](http://wiki.qtcentre.org/index.php?title=Designing_Custom_Controls_with_PyQt) for discussion and additions to the article and example code.

- [Low-Level Text Layouts](http://doc.trolltech.com/qq/qq24-textlayouts.html) - an article in Qt Quarterly 24 with downloadable example code written in C++, Java and Python.

- [Notes for embedding python in your C/C++ app.](http://pkaudio.blogspot.com/2008/11/notes-for-embedding-python-in-your-cc.html)

  - Patrick Stinson, the author of the above guide, has written it up on a Wiki here: [http://trac2.assembla.com/pkaudio/wiki/EmbeddedInterpreter](http://trac2.assembla.com/pkaudio/wiki/EmbeddedInterpreter)

- The [outline of a Wiki book](http://en.wikibooks.org/wiki/PyKDE_Programming) about PyKDE programming.

- The [Python Programming Wiki book](http://en.wikibooks.org/wiki/Python_Programming) contains a [chapter about PyQt4](http://en.wikibooks.org/wiki/Python_Programming/PyQt4).

- [Mini Sip Example](./PyQt(2f)MiniSipExample.html) is an introduction to using SIP to create bindings for your own libraries.
