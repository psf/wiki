## Historical Records 

These tutorials and resources are for Qt 3:

- [IBM Qt/PyQt tutorial](http://www-128.ibm.com/developerworks/linux/library/l-qt/?ca=dgr-linuxw02QTPyQT) - a toy application is developed, first with Tkinter, then with PyQt.

- [PyQT-tutorial](http://www.cs.usfca.edu/~afedosov/qttut/) - uses Qt Designer, very good for beginners.

- [Jonathan Gardner\'s PyQt tutorial](http://wiki.python.org/moin/JonathanGardnerPyQtTutorial) - writing a GUI for the `at`{.backtick} utility.

- [GUI Programming with Python: QT Edition](http://www.commandprompt.com/community/pyqt) - A very complete document about PyQt development (also mentioned in [Books](./PyQt(2f)Books.html))
