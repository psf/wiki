# LogicTools

::: {#content dir="ltr" lang="en"}
Tools for dealing with logic, logical inference, local programming, expert systems, etc in Python:

- [pyDatalog](https://sites.google.com/site/pydatalog/){.https} adds the logic programming paradigm to Python

- [Pychinko](http://www.mindswap.org/~katz/pychinko/){.http} is an RDF friendly forward-chaining rule engine
:::
