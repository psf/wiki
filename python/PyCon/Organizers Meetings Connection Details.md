# PyCon/Organizers Meetings Connection Details

::: {#content dir="ltr" lang="en"}
Pycon Organizers Meetings are held on IRC.

Connect to chat.freenode.net, and #pycon.

If you don\'t have a chat client, one way to do this is get [chatZilla](http://chatzilla.hacksrus.com/){.http}for Firefox, and in a Firefox adress tab type the URI:

- [irc:://chat.freenode.net/#pycon](irc:://chat.freenode.net/#pycon){.irc}
