# JustVanRossum

:::: {#content dir="ltr" lang="en"}
Just van Rossum is a Dutch (typo)graphic designer / [MacPython](MacPython) guru. Started the [/LettError](./JustVanRossum(2f)LettError.html){.nonexistent} virtual type foundry with [ErikVanBlokland](ErikVanBlokland), with whom he worked at [Meta Design](http://www.metadesign.com/){.http}. Fame came with the release of Beowolf (co-designed with [ErikVanBlokland](ErikVanBlokland)), a font whose ragged edges shift randomly each time you print the font. He\'s the developer of the [/MacPythonIDE](./JustVanRossum(2f)MacPythonIDE.html){.nonexistent}, designed the popular [Python Powered](http://starship.python.net/~just/pythonpowered/){.http} logo, developed [/DrawBot](./JustVanRossum(2f)DrawBot.html){.nonexistent} and [FontTools](./FontTools.html){.nonexistent}, helps out with [PyObjC](./PyObjC.html){.nonexistent} and [MacPython](MacPython), etc.

We\'ve started a page with notes about learning Cocoa using [/PyObjC](./JustVanRossum(2f)PyObjC.html){.nonexistent}: LtrWiki:StartCocoa
