------------------------------------------------------------------------

Current implementations of the specification (lets hope this list gets longer!):

- [Fnorb](http://sourceforge.net/projects/fnorb){.http}

- [OmniOrb](http://omniorb.sourceforge.net/){.http}

- [Orbit Python](http://sourceforge.net/projects/orbit-python){.http} *python ORB with no need for an IDL compiler* **this project is closed**

- [ftp://ftp.gnome.org/pub/gnome/sources/pyorbit/](ftp://ftp.gnome.org/pub/gnome/sources/pyorbit/){.ftp} orbit-python\'s successor for ORBit2
