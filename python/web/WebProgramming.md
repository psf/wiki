# Web Programming in Python 

This topic guide attempts to cover every aspect of programming Web applications (both clients and servers) using Python.

## Server Programming 

Topics concerned with writing providing Web applications, doing server-side programming, providing a Web site or pages which use Python in some way:

- [WebFrameworks](WebFrameworks) - for developing server-side Web applications in Python

- [CgiScripts](CgiScripts) - information on writing CGI scripts/programs in Python when the [WebFrameworks](WebFrameworks) seem like too much work or aren\'t available at your provider

- [WebServers](WebServers) - server solutions written in (or using) Python

- [ContentManagementSystems](ContentManagementSystems) - solutions specially designed for organising and publishing content

- [WebServices](WebServices) - tools for accessing and providing functionality via APIs over the Web

## Client Programming 

Topics concerned with accessing Web applications, sites or pages using Python:

- [WebBrowserProgramming](WebBrowserProgramming) - interfacing with existing browsers and browser technologies

- [WebClientProgramming](WebClientProgramming) - writing clients, typically at a lower level than a full Web browser

- [WebServices](WebServices) - tools for accessing and providing functionality via APIs over the Web

## Related Links 

Related topics of interest:

- [WebStandardisation](WebStandardisation) - working towards common solutions and APIs for Python Web programming

- [DatabaseProgramming](DatabaseProgramming) - guidance on available [DatabaseInterfaces](DatabaseInterfaces) and related tools

- [Templating](Templating) - generating output, messages and more from Web applications

- [BuildAnIntranet](BuildAnIntranet) - advice on how one might go about developing and deploying an intranet solution
