# Web Development

Python has been a web development language since the late 1990s, and the ecosystem shows it. This section covers frameworks, standards, community events, and reference materials from the Python web world. Much of it is archival -- reflecting the long history of Python on the web, from CGI scripts to full-stack frameworks.

## Overviews and Guides

- [WebProgramming](WebProgramming) -- broad topic guide covering server-side and client-side web programming in Python
- [WebFrameworks](WebFrameworks) -- comprehensive list of Python web frameworks, from full-stack to lightweight
- [WSGIImplementations](WSGIImplementations) -- catalog of WSGI servers and middleware implementations
- [WebProgrammingBooks](WebProgrammingBooks) -- books on Python web development (many older, some still relevant)
- [WebProgrammingShootOut](WebProgrammingShootOut) -- comparing frameworks by building the same app in each
- [WebProgrammingTemplate](WebProgrammingTemplate) -- blank template for documenting a web framework
- [WebProgrammingTemplateHelp](WebProgrammingTemplateHelp) -- how to fill out the framework documentation template

## Frameworks

- [Django](Django) -- the full-stack framework with batteries included, ORM, admin panel, and strong opinions
- [Flask](Flask) -- Werkzeug-based microframework with Jinja2 templating
- [CherryPy](CherryPy) -- object-oriented framework with its own HTTP server
- [TurboGears](TurboGears) -- full-stack "megaframework" aiming for rapid development
- [Quixote](Quixote) -- lightweight framework focused on simplicity
- [Albatross](Albatross) -- web application toolkit from Object Craft
- [Aquarium](Aquarium) -- another early Python web framework

## Twisted

- [Twisted](Twisted) -- overview and links for the Twisted networking engine
- [TwistedMatrix](TwistedMatrix) -- detailed page on Twisted as a scalable web server (not just a web framework)
- [Twisted-Examples](Twisted-Examples) -- code examples for Twisted
- [TwistedWebServerSampleOnWindows](TwistedWebServerSampleOnWindows) -- running Twisted as a Windows service
- [TwistedSprint](TwistedSprint) -- sprint event pages for Twisted development

## Zope and Plone

- [Zope](Zope) -- the Zope application server
- [ZopeBooks](ZopeBooks) -- books about Zope and Plone
- [ZopePageTemplates](ZopePageTemplates) -- Zope's template system for generating dynamic web pages
- [Plone](Plone) -- the Plone CMS built on top of Zope
- [PloneOrgMigration](PloneOrgMigration) -- sprint to migrate plone.org (EuroPython 2005)
- [PloneSeleniumSprint](PloneSeleniumSprint) -- sprint on Selenium testing for Plone
