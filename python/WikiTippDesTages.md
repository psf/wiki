# WikiTippDesTages

::: {#content dir="ltr" lang="en"}
- Auf [BenutzerEinstellungen](./BenutzerEinstellungen.html){.nonexistent} die Sprache auf \"Deutsch\" stellen, dann sind auch Emails in Deutsch

- Rechts oben auf [BenutzerEinstellungen](./BenutzerEinstellungen.html){.nonexistent} klicken, um eine Benutzerkennung zu erstellen und weitere Funktionen zu aktivieren.

- Als Benutzerkennung am besten einen [WikiNamen](./WikiNamen.html){.nonexistent} wählen ([VornameNachname](./VornameNachname.html){.nonexistent}).

- In den [BenutzerEinstellungen](./BenutzerEinstellungen.html){.nonexistent} kann man sich die Menüleiste individuell einrichten.

- In den [BenutzerEinstellungen](./BenutzerEinstellungen.html){.nonexistent} kann man die Größe des Editorfensters festlegen.

- Im Browser oder in den [BenutzerEinstellungen](./BenutzerEinstellungen.html){.nonexistent} die Sprache auf \"de\" stellen, dann sind die Wiki-Menüs und Hilfetexte auf Deutsch!

- In den Benutzereinstellungen kann man auch eigenes CSS definieren (beliebige URL!), wenn einem die Vorgabe nicht gefällt.

- Durch Setzen des Lesezeichens auf [AktuelleÄnderungen](./Aktuelle(c384)nderungen.html) sieht man leicht, was seit dem letzten Besuch geändert wurde.

- [GesuchteSeiten](GesuchteSeiten) zeigt [WikiNamen](./WikiNamen.html){.nonexistent}, zu denen es noch keine Seite gibt.

- [VerwaisteSeiten](VerwaisteSeiten) zeigt Seiten, die von nirgendwo per [WikiName](WikiName) referenziert werden.

- Die Verwendung eines sog. [WikiBadge](./WikiBadge.html){.nonexistent} hilft, den Überblick zu bewahren.

- Mit [CategoryCategory](CategoryCategory) bekommt man eine Übersicht über verfügbare Kategorien.

- Ein [WikiWord](./WikiWord.html){.nonexistent} ist ein einzelnes Teilwort eines [WikiNamen](./WikiNamen.html){.nonexistent}. Ein [WikiName](WikiName) ist ein aus mehreren vorne groß geschriebenen Worten zusammengesetztes Wort.

- [SeitenName](./SeitenName.html){.nonexistent}n ([WikiName](WikiName)n) IMMER in der Einzahl verwenden, sonst entstehen mehrere Seiten zu einem Thema!

- Man darf auf Wikiseiten und auch in den Seitennamen Umlaute verwenden.

- Mit \"@ SIG @\" (ohne Leerzeichen) unterschreibt man einen Text.

- Traut Euch, Inhalte hinzuzufügen - man kann nichts kaputtmachen, denn die Seiten werden automatisch gesichert!

<!-- -->

- Mit [SpellCheck](./SpellCheck.html){.nonexistent} kann man eine Rechtschreibprüfung auslösen. Wichtig: unbekannte Worte nur dann hinzufügen, wenn man sich 100%ig sicher ist, dass sie richtig geschrieben sind.

- Mit \"[meinedatei.txt]( "Upload new attachment "meinedatei.txt""){.attachment .nonexistent}\" kann man einen Verweis auf einen Dateianhang erzeugen (und den dann auch zum Hochladen der Datei benutzen).

- Email-Adressen einfach eintippen ([meine.mail@gmx.de](mailto:meine.mail@gmx.de){.mailto}) - Wiki macht dann automatisch einen anklickbaren Link daraus.

- Web-Adressen einfach eintippen ([http://wo.auch.immer.de/](http://wo.auch.immer.de/){.http}), Wiki macht automatisch einen anklickbaren Link daraus.

- Bilder werden automatisch eingefügt, wenn man eine URL (oder ein attachment) auf ein Bild eingibt.

- Keine neuen, sehr kleinen Seiten für Unterthemen anlegen, wenn das Thema auch erschöpfend in wenigen Zeilen auf einer schon vorhandenen, passenden Seite behandelt werden kann.

- Wikis sind toll - vor allem wenn man die Hilfe gelesen hat. 8)

- [WikiWiki](./WikiWiki.html){.nonexistent} = hawaiianisch für \"schnell\".

- Mit der Lupe kann man Titel oder Volltext durchsuchen und auch reguläre Ausdrücke verwenden.

- Mit dem Brief kann man Seiten abonnieren, so dass man bei Änderungen per Email benachrichtigt wird.

- Die blaurote Brille zeigt Änderungen hervorgehoben an, so dass man nicht alles durchlesen muss.

- Mit dem Info-Icon bekommt man die Lebensgeschichte der aktuellen Seite, also Zugriff auf alte Revisionen.

- [TitelIndex](TitelIndex) ist ein normaler Index aller Seiten, [WortIndex](WortIndex) führt jedes Teilwort eines Seitennamen auf.

- Man kann auch andere Seiten als die [StartSeite](StartSeite) als persönliche Einstiegsseite benutzen, z.B. [AktuelleÄnderungen](./Aktuelle(c384)nderungen.html) oder die eigene Homepage.
