# PyObjC {#PyObjC}

The [/PyObjC](./MacPython(2f)PyObjC(2f)PyObjC.html){.nonexistent} project aims to provide a bridge between the Python and Objective-C programming languages. The bridge is intended to be fully bidirectional, allowing the Python programmer to take full advantage of the power provided by various Objective-C based toolkits and the Objective-C programmer transparent access to Python based functionality. What this really means is that you can write full fledged [\"Cocoa\"](http://developer.apple.com/cocoa){.http} applications in pure Python.

Currently developed by [RonaldOussoren](RonaldOussoren), [BillBumgarner](BillBumgarner), [SteveMajewski](./SteveMajewski.html){.nonexistent}, [JustVanRossum](JustVanRossum), [JackJansen](JackJansen), and others.
