# MacPython/Packages

::: {#content dir="ltr" lang="en"}
For Pre-built packages for Python 2.5, see:

- [http://pythonmac.org/packages/py25-fat/](http://pythonmac.org/packages/py25-fat/){.http}

For Pre-built packages for Python 2.4, see:

- [http://pythonmac.org/packages/py24-fat/](http://pythonmac.org/packages/py24-fat/){.http}

These packages are built to work with the universal binaries available from python.org. They should work with OS-X 10.3.9, 10.4.\* and 10.5.\*

**NOTE:** This repository is not being well maintained at this point. However, many package maintainers are now supportin OS-X themselves, so check the web site for dowloadable binaries, and/or try easy_install

For info and discussion about Packages for the new Universal Framework Build of Python 2.4.3, see: [/UniversalPackages](./MacPython(2f)Packages(2f)UniversalPackages.html){.nonexistent}
:::
