# /FontTools {#A.2FFontTools}

TTX is a tool to convert [/OpenType](./MacPython(2f)FontTools(2f)OpenType.html){.nonexistent} and [/TrueType](./MacPython(2f)FontTools(2f)TrueType.html){.nonexistent} fonts to and from XML. [/FontTools](./MacPython(2f)FontTools(2f)FontTools.html){.nonexistent} is a library for manipulating fonts, written in Python. It supports [TrueType](./TrueType.html){.nonexistent}, [OpenType](./OpenType.html){.nonexistent}, AFM and to an extent Type 1 and some Mac-specific formats. Developed by [JustVanRossum](JustVanRossum).

# status

[FontTools](./FontTools.html){.nonexistent} 2.0b1 was released on September 12, 2002.
