# MacPython/IrcChannel

::: {#content dir="ltr" lang="en"}
The \"official\" chat room for the [MacPython](MacPython) community is the [#macpython](irc://irc.freenode.net/#macpython){.irc} channel on the irc.freenode.net network. This IRC channel supersedes the old [MacPython](MacPython) AIM/iChat chat room.

Topics:

- [MacPython](MacPython)

- [../PyObjC](./MacPython(2f)PyObjC.html)

- newbie questions

- dev questions

If you\'re new to IRC, or are looking for a nice Cocoa IRC client, we highly recommend [Colloquy](http://colloquy.info/){.http}.
