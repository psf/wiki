# MacPython/plistservices

::: {#content dir="ltr" lang="en"}
Reads and writes [CoreFoundation](./CoreFoundation.html){.nonexistent} XML Property List files, and is actually used to generate [http://sarwat.net/opensource/](http://sarwat.net/opensource/){.http}. Includes an ISO 8601 string parser conforming to [w3c](http://www.w3.org/TR/NOTE-datetime){.http}'s note. plistservices is more functional than the version of [../plistlib](./MacPython(2f)plistlib.html){.nonexistent} included with Python 2.3 and has an interface similar to Foundation\'s \[[http://developer.apple.com/documentation/Cocoa/Reference/Foundation/ObjC_classic/Classes/NSPropertyListSerialztion.html](http://developer.apple.com/documentation/Cocoa/Reference/Foundation/ObjC_classic/Classes/NSPropertyListSerialztion.html){.http} NSPropertyListSerialization\]
