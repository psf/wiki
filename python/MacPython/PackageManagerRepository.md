# What Is PackageManager? {#What_Is_PackageManager.3F}

Since Python 2.3, [MacPython](MacPython) has included the [/PackageManager](./MacPython(2f)PackageManagerRepository(2f)PackageManager.html) application. Though [/PackageManager](./MacPython(2f)PackageManagerRepository(2f)PackageManager.html) is still in its early stages, it\'s a huge time saver for acquiring Python modules and extensions that are tested (or modified) to work with OS X. The majority of modules in a [/PackageManagerRepository](./MacPython(2f)PackageManagerRepository(2f)PackageManagerRepository.html){.nonexistent} can be installed precompiled with a single click.

Note that [/PackageManager](./MacPython(2f)PackageManagerRepository(2f)PackageManager.html) is **only** for [MacPython](MacPython) 2.3. The package manager page is located at [http://www.python.org/packman/](http://www.python.org/packman/){.http}.

# How Do I Use PackageManager? {#How_Do_I_Use_PackageManager.3F}

With an normal installation of [MacPython](MacPython) 2.3, [/PackageManager](./MacPython(2f)PackageManagerRepository(2f)PackageManager.html) is located at [/ApplicationsMacPython](./MacPython(2f)PackageManagerRepository(2f)ApplicationsMacPython.html){.nonexistent}-2.3/PackageManager.app. By default, it will point to the official [PackageManagerRepository](./PackageManagerRepository.html){.nonexistent}, maintained by [JackJansen](JackJansen). You may choose an alternate [/PackageManagerRepository](./MacPython(2f)PackageManagerRepository(2f)PackageManagerRepository.html){.nonexistent} by choosing Open URL from the File menu.

# Where Are The Alternate Package Manager Repositories? {#Where_Are_The_Alternate_Package_Manager_Repositories.3F}

So far, there is only one known alternate [/PackageManagerRepository](./MacPython(2f)PackageManagerRepository(2f)PackageManagerRepository.html){.nonexistent} for OS X (or any platform, for that matter). This \"Official Unofficial\" [/PackageManagerRepository](./MacPython(2f)PackageManagerRepository(2f)PackageManagerRepository.html){.nonexistent} is maintained by [BobIppolito](BobIppolito), and you can find instructions and an up to date package list at [http://undefined.org/python/pimp/](http://undefined.org/python/pimp/){.http}. It includes many more modules than the official [/PackageManagerRepository](./MacPython(2f)PackageManagerRepository(2f)PackageManagerRepository.html){.nonexistent} inluding some compiled in exotic ways (such as Numeric against Apple\'s vecLib, or PIL with the libtiff patch).
