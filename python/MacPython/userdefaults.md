# MacPython/userdefaults

::: {#content dir="ltr" lang="en"}
[UserDefaults](./UserDefaults.html){.nonexistent} is an interface for persistent user defaults (user preferences) for Python programs. [UserDefaults](./UserDefaults.html){.nonexistent} provides an application defaults dictionary that can store [CoreFoundation](./CoreFoundation.html){.nonexistent}-style property list objects. userdefaults consists of a pure python module that uses a back-end extension module to access the operating system\'s preferences API.
