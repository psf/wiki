# PythonSound

::: {#content dir="ltr" lang="en"}
The Python Sound Project aims to develop a productive community around Python, Csound and other synthesis engines as tools for algorithmic composition of electroacoustic music. To join the mailing list, send an empty message to:

[python-sound-subscribe@googlegroups.com](mailto:python-sound-subscribe@googlegroups.com){.mailto}

or check:

[http://groups.google.com/group/python-sound](http://groups.google.com/group/python-sound){.http}
