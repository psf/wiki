# EpyDoc

::: {#content dir="ltr" lang="en"}
[EpyDoc](http://epydoc.sourceforge.net/){.http} is a documentation generator for Python similar to [JavaDoc](./JavaDoc.html){.nonexistent}. It provides its own simple text-markup language, but also supports the use of [reStructuredText](reStructuredText). It does not support Zope \"scarecrow\"-style interfaces.

Development of [EpyDoc](http://epydoc.sourceforge.net/){.http} is hosted on [SourceForge](SourceForge).
