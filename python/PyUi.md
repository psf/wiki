# PyUi {#PyUi-1}

Specialized GUI Toolkit for rendering GUIs in Opengl. The widgets themselves are OpenGL elements.

Suitable for game programming. Easy to use.

[PyUi homepage](http://pyui.sourceforge.net/){.http}

# Dependencies {#Dependencies}

[PyGame](PyGame), [PyOpenGl](./PyOpenGl.html), PIL ([PythonImagingLibrary](PythonImagingLibrary))

# Caveats {#Caveats}

Last release: PyUI version 1.0 for Python 2.2 on the 2003-08-17
