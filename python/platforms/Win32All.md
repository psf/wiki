[PyWin32](PyWin32) (previously \"Win32All\") is a package that includes extensions for accessing the Win32 API, Windows COM bindings (e.g. modules like \"win32api\", \"win32com\", etc.) and the [PythonWin](PythonWin) IDE.

Mark Hammond created these extensions, and later maintained them while working at [ActiveState](ActiveState). These extensions are now hosted at [Sourceforge](https://sourceforge.net/projects/pywin32/).

Information on the Python-win32 mailing list can be found [here](http://mail.python.org/mailman/listinfo/python-win32).
