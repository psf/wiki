# DrScheme

::: {#content dir="ltr" lang="en"}
DrScheme is an interactive, integrated, graphical programming environment for the Scheme, MzScheme, and MrEd programming languages. DrScheme runs under Windows (95 and up), Mac OS, and Unix/X. It is the inspiration for the [DrPython](DrPython) editor.

See: [http://www.drscheme.org/](http://www.drscheme.org/){.http}
:::
