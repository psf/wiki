# Virtualenv

::: {#content dir="ltr" lang="en"}
Tool for working with your Python packages in isolation:

- Docs: [http://www.virtualenv.org/](http://www.virtualenv.org/){.http}

- Development: [https://github.com/pypa/virtualenv](https://github.com/pypa/virtualenv){.https}

There is also `virtualenvwrapper`{.backtick} that extends `virtualenv`{.backtick} to make it easier to work with multiple virtualenvs:

- Docs: [http://www.doughellmann.com/projects/virtualenvwrapper/](http://www.doughellmann.com/projects/virtualenvwrapper/){.http}

- Development: [https://bitbucket.org/dhellmann/virtualenvwrapper](https://bitbucket.org/dhellmann/virtualenvwrapper){.https}

The ability is also included in Python 3.3 through [PEP 405](http://www.python.org/dev/peps/pep-0405/){.http}.
:::
