IRC stands for Internet Relay Chat and you need to search Google if you want to know more about it. Here is some information related to IRC and Python.

## Python IRC Channels {#Python_IRC_Channels}

Are mostly on [freenode](http://freenode.net){.http} IRC network:

- [irc://chat.freenode.net/python](irc://chat.freenode.net/python){.irc} - official web page at [http://pound-python.org/](http://pound-python.org/){.http}

- [irc://chat.freenode.net/macpython](irc://chat.freenode.net/macpython){.irc} - see [MacPython/IrcChannel](./MacPython(2f)IrcChannel.html) \|\|
