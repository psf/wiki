# PyLaunchy {#PyLaunchy-1}

[PyLaunchy](http://pylaunchy.sourceforge.net){.http} is an extension to [Launchy\'s](http://www.launchy.net){.http} plugin system, that allows developing Launchy plugins in the [Python](Python) programming language.

[Launchy](http://www.launchy.net){.http} is a free open source application launcher for GNU/Linux and Windows, written in C++ and uses QT. The application can be extended by writing C++ plugins

PyLaunchy provides a framework for writing Launchy plugins as Python scripts. It uses [boost.python](./boost(2e)python.html) for C++ bindings and [PyQt4](PyQt4) for integration with QT.
