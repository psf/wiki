# PyLint

::: {#content dir="ltr" lang="en"}
[PyLint](http://www.logilab.org/projects/pylint){.http} is a [static source analysis](./static(20)source(20)analysis.html) tool, available within such development tools [as PyDev](http://pydev.sourceforge.net/pylint.html){.http}, for instance.

# Availability {#Availability}

\* Currently runs on Python 2.4 and up

\* An [effort](./PyLint(2d)3k.html) is getting underway to port to Python 3.x, while preserving Python 2.5 and up compatibility
:::
