## Bücher 

- [Routineaufgaben mit Python automatisieren](https://dpunkt.de/produkt/routineaufgaben-mit-python-automatisieren-2/) Die deutsche Übersetzung () des Buches \"Automate the Boring Stuff with Python\" von Al Schweigart in der 2. Auflage (jetzt für Python 3) ist eine Python-Einführung am Beispiel von alltäglichen Aufgaben eines Computernutzers. Al Schweigart stellt die [englische Originalausgabe](https://automatetheboringstuff.com/) kostenlos zum online-Lesen zur Verfügung. [dpunkt.verlag](https://dpunkt.de/).

- [Python 3 Das umfassende Handbuch](http://openbook.rheinwerk-verlag.de/python/) von Johannes Ernesti und Peter Kaiser, als Openbook aus dem [Rheinwerk Verlag](https://www.rheinwerk-verlag.de/). Alternativ auch als Buch/eBook gegen Bezahlung ![;-)](/wiki/europython/img/smile4.png ";-)")

- [Python von Kopf bis Fuß](https://oreilly.de/produkt/python-von-kopf-bis-fuss/) von Paul Barry ist ein Buch für Einsteiger. [O\'Reilly Germany](http://www.oreilly.de/).

- [Programmieren lernen mit Python](https://oreilly.de/produkt/programmieren-lernen-mit-python/) von Allen B. Downey ist an Einsteiger gerichtet. Die 2. Auflage behandelt Python 3.4.

- [A Byte of Python (deutsche Übersetzung)](http://cito.github.io/byte_of_python/) ist die deutsche Übersetzung von Swaroop C Hs [A Byte of Python](http://www.swaroopch.com/notes/Python) für Python 2.x.

- [Einstieg in Python](https://www.rheinwerk-verlag.de/einstieg-in-python-ideal-fuer-programmiereinsteiger/), von Thomas Theis, ist ein deutsches Buch für Anfänger. [Rheinwerk Verlag](https://www.rheinwerk-verlag.de/).

- [Python kurz & gut](https://oreilly.de/produkt/python-kurz-gut/) ist eine Übersetzung von \"Python Pocket Reference\" (Mark Lutz) von Stefan Fröhlich und ist auf dem Stand von Python 2.7 und 3.4. [O\'Reilly Germany](http://www.oreilly.de/).

- [Programmieren lernen](http://www.alan-g.me.uk/tutor/german/index.htm) ist eine Übersetzung (durch Bruno Schäfer) von Alan Gaulds [Learning to Program](http://www.alan-g.me.uk/l2p/).

- [Python für Kids](http://python4kids.net) - Gregor Lingls Buch, das Kinder in der Mittel- und Oberstufe an Python heranführt. Die letzte, 4. Auflage benutzt Python 3.1.

- [Python & XML](http://www.oreilly.de/catalog/pythonxmlger/) ist eine Übersetzung von \"Python and XML\" (Christopher A. Jones & Fred L. Drake, Jr.) von Dinu Gherman. [O\'Reilly Germany](http://www.oreilly.de/).

- [Einführung in Python 3](https://www.hanser-fachbuch.de/buch/Einfuehrung+in+Python+3/9783446452084) ist ein Buch von [Bernd Klein](http://www.bklein.de), was besonders geeignet ist für Programmieranfänger, aber auch für Umsteiger von anderen Programmiersprachen. [Hanser Verlag](http://www.hanser-fachbuch.de).

- [Hitchhiker's Guide für Python](https://oreilly.de/produkt/hitchhikers-guide-fuer-python/) Kenneth Reitz und Tanya Schlusser befasst sich mit Python Best Practices. [O\'Reilly Germany](http://www.oreilly.de/).
