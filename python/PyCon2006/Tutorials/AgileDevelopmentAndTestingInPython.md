## Agile development and testing in Python {#Agile_development_and_testing_in_Python}

### Summary {#Summary}

We will present a Python application that we developed together as an \"agile team\", using agile development and testing approaches, techniques and tools. The value of the tutorial will consist on one hand in detailing the development and testing methodologies we used, and on the other hand in demonstrating specific Python tools that we used for our development and testing. We will cover TDD, unit testing, code coverage, functional/acceptance testing, Web application testing, continuous integration, source code management, issue tracking, project management, documentation, Python package management.

### Outline {#Outline}

1st hour:

- Intro: agile development and testing concepts

- Source control management ([subversion](http://subversion.tigris.org/){.http}) and issue tracking/project management ([Trac](http://www.edgewall.com/trac/){.http})

- TDD/unit testing ([py.test](http://codespeak.net/py/current/doc/test.html){.http}, [doctest](http://docs.python.org/lib/module-doctest.html){.http}, [nose](http://somethingaboutorange.com/mrl/projects/nose/){.http}, [TestOOB](http://testoob.sourceforge.net/){.http})

- unit tests as documentation ([doctest and epydoc](http://agiletesting.blogspot.com/2005/02/agile-documentation-with-doctest-and.html){.http})

2nd hour:

- code coverage (the [coverage](http://www.nedbatchelder.com/code/modules/coverage.html){.http} module)

- source code analysis ([pylint](http://www.logilab.org/projects/pylint){.http}, [pyflakes](http://divmod.org/projects/pyflakes){.http})

- acceptance/functional testing ([PyFit/FitNesse](http://agiletesting.blogspot.com/2004/11/writing-fitnesse-tests-in-python.html){.http}, [TextTest](http://texttest.org/){.http})

- performance testing ([FunkLoad](http://funkload.nuxeo.org/){.http}, [twill](http://www.idyll.org/%7Et/www-tools/twill.html){.http})

3rd hour:

- Web application testing with [Selenium](http://confluence.public.thoughtworks.org/display/SEL/Home){.http} and twill

- Scripting Selenium tests using a [Twisted-based server](http://agiletesting.blogspot.com/2005/03/web-app-testing-with-python-part-2.html){.http}

- Python package management ([distutils](http://docs.python.org/lib/module-distutils.html){.http}, [setuptools](http://peak.telecommunity.com/DevCenter/setuptools){.http}, [Cheesecake index](http://pycheesecake.org){.http})

- Continuous integration and \'smoke test\' ([buildbot](http://buildbot.sourceforge.net/){.http})
