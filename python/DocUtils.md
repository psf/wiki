# DocUtils

::: {#content dir="ltr" lang="en"}
[Docutils](http://docutils.sourceforge.net/){.http} is an open-source text processing system for processing plaintext documentation into useful formats, such as HTML or LaTeX. It includes [reStructuredText](http://docutils.sourceforge.net/rst.html){.http}, the easy to read, easy to use, what-you-see-is-what-you-get plaintext markup language.
