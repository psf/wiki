Under \"Bookmarks,\" pick \"Manage Bookmarks\" \# \# 2. Put the bookmark in the \"Quick Searches\" folder. \# \# \# Now, at the address bar, you can just type something like: \# \# python cgi \# \# \...and you\'ll be at the Python Reference CGI page. \# \# Or type something like: \# \# python [FrontPage](FrontPage) \# \# \...and you\'ll be at the front page of the wiki. \# \# Take care! \# \# \# (If you\'re curious how this works, it\'s all just a bunch \# of webservices and browser redirection.) \# \# ( You can pull down the namespace binding with the following \# code, if you like, as well: \# \# import xmlrpclib \# nameserver = xmlrpclib.[ServerProxy](./ServerProxy.html){.nonexistent}( \"[http://services.taoriver.net:9089/](http://services.taoriver.net:9089/){.http}\" ) \# nameserver.parse_description( \"[http://wrap.localnames.taoriver.net/spaces/pythoninfo/](http://wrap.localnames.taoriver.net/spaces/pythoninfo/){.http}\" ) \# \# You\'ll get back a big fat dictionary of name-URL bindings. \# See [http://services.taoriver.net:9089/](http://services.taoriver.net:9089/){.http} for other functionality available to you. ) \#
