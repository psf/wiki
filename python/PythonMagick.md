# PythonMagick

::: {#content dir="ltr" lang="en"}
PythonMagick provides object oriented bindings for the [ImageMagick](ImageMagick) Library.

[ImageMagick](ImageMagick) can be found at [http://www.imagemagick.org/](http://www.imagemagick.org/){.http}

PythonMagick is available under the API link from [ImageMagick](ImageMagick)
:::
