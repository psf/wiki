# PyPy 

PyPy is a Python implementation in Python. Yes, it can interpret itself!

Please refer to the [PyPy home page](http://pypy.org) for more information about this Python implementation.

It includes a JIT compiler, stackless-like functionality, uses the [Boehm garbage collector](http://www.hpl.hp.com/personal/Hans_Boehm/gc/) and has whole lot of \"backends\" - C, LLVM, JavaScript, CLI, Java.
