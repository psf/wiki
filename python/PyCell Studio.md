# PyCell Studio

::: {#content dir="ltr" lang="en"}
[PyCell](./PyCell.html){.nonexistent} Studio is a complete, stand-alone environment for creating universal [OpenAccess](./OpenAccess.html){.nonexistent} PCells and layout generators. [PyCells](./PyCells.html){.nonexistent} operate within any [OpenAccess](./OpenAccess.html){.nonexistent} tool, including Cadence Virtuoso, Silicon Canvas Laker, and Mentor Calibre.

[PyCells](./PyCells.html){.nonexistent} are authored using Python and the Ciranova Layout API, accessed as a Python extension package.

[PyCell](./PyCell.html){.nonexistent} Studio is available for free download (after registration) at the Ciranova web site: [http://www.synopsys.com/Tools/Implementation/CustomImplementation/Pages/pycell-studio.aspx](http://www.synopsys.com/Tools/Implementation/CustomImplementation/Pages/pycell-studio.aspx){.http}
