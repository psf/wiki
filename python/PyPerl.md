# PyPerl

::: {#content dir="ltr" lang="en"}
pyperl is a module for integrating Perl in Python.

See:

- [PyPerl at ASPN](http://aspn.activestate.com/ASPN/Modules/dist_html?dist_id=10617){.http}

- [perlmodule documentation at ASPN](http://aspn.activestate.com/ASPN/CodeDoc/pyperl/perlmodule.html){.http}

- [Download latest PyPerl module](http://downloads.activestate.com/Zope-Perl/){.http}

- pyperl is currently unmaintained. There is an unofficial version distributed by Felix Schwarz named [1.0.1c](http://www.felix-schwarz.name/files/opensource/pyperl/){.http} which fixes some compilation errors on newer systems.

- [pyperl: Perl & Python is True!](http://conferences.oreillynet.com/cs/os2001/view/e_sess/1437){.http}

- [original PyPerl 1.0 distribution on the CPAN](http://search.cpan.org/dist/pyperl/){.http}
