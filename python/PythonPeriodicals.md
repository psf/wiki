# PythonPeriodicals

::: {#content dir="ltr" lang="en"}
Several periodicals dedicated to the Python Programming language (some active, some not, some are archives).

In no particular order:

## Active {#Active}

- [Python Weekly](http://pythonweekly.com/){.http} - Started in mid 2013 (estimated), delivered via email.

- [Pycoders\' Weekly](http://pycoders.com/){.http} - Launched Feb 2012; issued weekly via email, with an archive.

## Inactive {#Inactive}

- [The Python Papers](http://pythonpapers.org/){.http} - Established in 2006, with free downloads of printable issues. Please consider contributing an article to this volunteer-run publication!

- [Python Magazine](http://www.pythonmagazine.com/){.http} - First issue came out in October 2007. Available as PDF. Subscription required, but with some issues downloadable at no cost.

- [The Python: Rag](http://www.pythonrag.org/){.http} - First issue came out in July 2009. Available as PDF. Free to download.

## Archived {#Archived}

- [PyZine](http://www.pyzine.com/){.http} - Published from 2002 to 2005, the archived articles are now available for viewing online without a subscription.

- [Python Journal](http://pythonjournal.cognizor.com/){.http} - Published around 2002, the archived articles are available online.
:::
