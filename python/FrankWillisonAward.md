# FrankWillisonAward

::: {#content dir="ltr" lang="en"}
The Frank Willison Award for Contributions to the Python Community is presented by O\'Reilly & Associates to a notable member of the Python community at each year\'s OSCON conference. Recipients of the award can be seen [on the python.org site](https://www.python.org/community/awards/frank-willison/){.https}
:::
