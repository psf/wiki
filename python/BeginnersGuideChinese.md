# Python初学者手册 {#Python.2BUh1bZoAFYktRjA-}

- 英文版在此: [http://wiki.python.org/moin/BeginnersGuide](http://wiki.python.org/moin/BeginnersGuide){.http}

没玩过编程？ Python语言是免费的，如果你知道从何处开始，它很容易上手！ 本指南将帮助你快速入门。

## 不了解Python？ {#A.2BTg1Ohonj-Python.2B.2Fx8-}

先读一下[Python简介](./BeginnersGuide(2f)OverviewChinese.html)：通过一个简短的介绍了解Python是什么。

## 获取Python {#A.2Bg7dT1g-Python}

接下来，在你的电脑上安装Python的解释器。

它用来读取Python程序并执行指令：在进行任何Python编程前你需要准备好它。

目前Python有两个主要版本：Python 2和Python 3。根据[Python2orPython3](Python2orPython3) 页面提供的建议，你可以决定哪个版本最适合你。在撰写本文时（2010年6月21号），以下内容假设你决定使用Python 2。

查看文章 [BeginnersGuide/Download](./BeginnersGuide(2f)Download.html)，按步骤下载正确的Python版本。

在某些阶段，你想编辑并保存你的程序代码，可以看一下 [HowToEditPythonCode](HowToEditPythonCode) 以听取一些意见和推荐方法。

## 学习Python {#A.2BW2ZOYA-Python}

接下来，阅读教程，并尝试用你的新Python解释器做一些简单的试验。

- 如果你之前没有编程经验，读一下 [编程新手](./BeginnersGuide(2f)NonProgrammersChinese.html)， 里面有适合你的教程列表。

- 如果你之前有编程经验，参考 [BeginnersGuide/Programmers](./BeginnersGuide(2f)Programmers.html), 里面列出了更多高级教程。

- 如果英语不是你的母语，阅读翻译好的教程更适合你。参考 python.org\'s [非英语资源列表](http://wiki.python.org/moin/Languages){.http}.

绝大部分教程都假设你知道如何在你的电脑上运行程序。

如果你使用Windows操作系统，可以参考[我如何在Windows平台上运行程序](http://www.python.org/doc/faq/windows/#how-do-i-run-a-python-program-under-windows){.http}以获得帮助

当你读完教程后，可以浏览[Python在线文档](http://docs.python.org){.http}。它包括：有用的[a tutorial](http://docs.python.org/tut/){.http}，[a Library Reference](http://docs.python.org//lib/){.http}详列了Python的标准库中的模块和\[\[[http://docs.python.org/ref/\|the](http://docs.python.org/ref/%7Cthe){.http} Language Reference\]介绍了Python语法的完整解释（有些枯燥）

在准备写第一个程序前，你需要一个文本编辑器。开始时使用任何你熟悉的编辑器（用记事本也可以）-但当你有经验后，你会想使用带有Python编程辅助功能的文本编辑器。参考 [PythonEditors](PythonEditors) ，里面包含了一个软件列表，介绍各种方便Python代码编辑功能的编辑器软件。

## 需要帮助吗? {#A.2BlwCJgV4uUqlUFw.3F}

需要一些帮助吗？加入邮件列表和新闻组，阅读 [BeginnersGuide/Help](./BeginnersGuide(2f)Help.html)。

绝大多数的Python书籍会包含Python语言的介绍：请查看 [IntroductoryBooks](IntroductoryBooks)中的建议。

参考[BeginnersGuide/Examples](./BeginnersGuide(2f)Examples.html)，里面的简单程序和代码片段会对你的学习有帮助。

或者，如果你喜欢通过听课的方式学习Python，你可以参加培训课或请一位培训老师到你公司上课。

参考 [PythonEvents](PythonEvents) 页面看在当地有无定期的培训课程，在[PythonTraining](PythonTraining)页面有培训老师列表。

老师可以加入 [EDU-SIG](http://www.python.org/sigs/edu-sig/){.http}，在该邮件列表中讨论如何教从K-12到大学的各种水平的学生来应用Python。

## 想寻找具体的Python模块或程序？ {#A.2BYPNb.2B2J.2BUXdPU3aE-Python.2BaiFXV2IWegtej.2F8f-}

- 首先参考 [Python Package Index](http://pypi.python.org/pypi){.http}.

- 如果在目录中找不到相关的内容，参考 [searching python.org](http://www.python.org/search/){.http} -这里你可以找到Python网站上提供的全部内容，包括[常见问题](http://www.python.org/doc/faq/){.http}，或者新闻组。详情：[在哪里查找](http://www.python.org/search/){.http}

- 接下来，尝试 [Google](http://www.google.com){.http}或其它你常用的搜索引擎。搜索"python"或其它关键字通常可以找到有用的信息。

- 最后，（注：如果实在搞不定了）你可以在comp.lang.python 用户组中发帖寻求帮助。

## 如何做出贡献? {#A.2BWYJPVVBaUfqNIXMu.3F}

- Python来源于[Python Software Foundation](http://www.python.org/psf/){.http}，该非营利组织拥有其版权。[向PSF捐款](http://www.python.org/psf/donations/){.http}在美国是免税的，你可能通过信用卡或[PayPal](http://www.paypal.com){.http}来捐助。

- 通过[Python bug跟踪](http://bugs.python.org/){.http}向Python核心组报告bug。

- 想向python核心组提供bug修复或补丁，先阅读[Python开发者指南](http://www.python.org/dev/){.http}来获取关于Python开发过程的更多信息。

- 想编辑Python官方文档[Python文档](http://www.python.org/doc/){.http}，请加入[Documentation SIG\]，写信给docs@python.org或者使用\[\[http://bugs.python.org/](http://www.python.org/sigs/doc-sig/){.http}增加文档修改内容。

- 想在Python社区发布你的模块或程序，使用[comp.lang.python.公告](news:comp.lang.python.announce){.news}。更多信息可以参考[Python邮件列表指南](http://www.python.org/community/lists/#comp-lang-python-announce){.http}

- 想给Python核心组提出修改意见，请将你的想法发到[comp.lang.python](news:comp.lang.python){.news}上。如果你已经有了代码实现，请遵照[Python补丁指南](http://www.python.org/patches/){.http}
