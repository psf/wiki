# SetuptoolsFeatures

::: {#content dir="ltr" lang="en"}
The goal is to fill the gap for setuptools users that want to move to packaging.

\- a develop command (Python bug [8668](http://bugs.python.org/issue8668 "Issue"){.interwiki}, [wiki page](./Distutils(2f)Proposals(2f)DevelopmentDistributions.html))

\- entry points (see [blog post](http://tarekziade.wordpress.com/2009/05/01/basic-plugin-system-using-abcs-and-the-extensions-package/){.http} and [code](https://bitbucket.org/tarek/extensions){.https})
:::
