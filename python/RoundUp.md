# Roundup

::: {#content dir="ltr" lang="en"}
Roundup is a simple-to-use/install issue-tracking system with command-line, web and e-mail interfaces. It is based on the winning design from [KaPingYee](KaPingYee) in the [SoftwareCarpentry](./SoftwareCarpentry.html){.nonexistent} \"Track\" design competition.

It\'s a [SourceForge](SourceForge) project, with the lead developer being [RichardJones](RichardJones). You can visit the project page at [http://roundup.sf.net/](http://roundup.sf.net/){.http} for more information.

## Roundup for python.org {#Roundup_for_python.org}

Roundup is used for tracking Python bugs at python.org The following pages accumulate various information about its installation and usage specifics in Python development:
