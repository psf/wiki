# Asking for Help: Is there any cross-platform GUI toolkit available for Python 3 apart from Tkinter? {#Asking_for_Help:_Is_there_any_cross-platform_GUI_toolkit_available_for_Python_3_apart_from_Tkinter.3F}

Hello sir is there any Cross platform GUI toolkit available for python 3? except tkinter.

------------------------------------------------------------------------

Take a look at the [Python2orPython3](Python2orPython3) page and also the [Python3PortingStatus](Python3PortingStatus) page. The latter links to the following resources:

- [http://getpython3.net/](http://getpython3.net/){.http} (doesn\'t seem to be working as of 2012-01-06)

- [http://pypi.python.org/pypi?:action=browse&c=533](http://pypi.python.org/pypi?:action=browse&c=533){.http} (packages tagged with Python 3 on PyPI)

I suggest browsing the latter of the above resources and selecting things in the \"Environment\" section. By selecting [X11 Applications](http://pypi.python.org/pypi?:action=browse&c=24&c=533){.http}, I can see that PyQt, for example, is available for Python 3. \-- [PaulBoddie](PaulBoddie) 2012-01-06 16:45:58
