# Education

Python has a long history in education, from introductory CS courses at universities to after-school programs for younger students. This section collects pages about the Python Education Special Interest Group (Edu-SIG), PyCon education events, and lists of schools that have adopted Python in their curricula.

## Edu-SIG

- [EduSig](EduSig) — the Python Education Special Interest Group, a community focused on Python in teaching
- [EduSig](EduSig/index) — Edu-SIG subpages with resources, data sets for exercises, and discussion

## Events & Meetings

- [EducationSummit](EducationSummit) — the PyCon Education Summit, with links to sessions and open spaces
- [EducationBof](EducationBof) — Birds of a Feather session on Python education at PyCon
- [TeachingPythonBof](TeachingPythonBof) — BoF specifically about teaching with Python

## Resources

- [SchoolsUsingPython](SchoolsUsingPython) — directory of high schools and universities using Python in their courses
- [EducationalCd](EducationalCd) — an educational CD distribution bundling Python and learning materials
- [EducationalSprint5MinuteIntroduction](EducationalSprint5MinuteIntroduction) — a quick intro to participating in an educational sprint
