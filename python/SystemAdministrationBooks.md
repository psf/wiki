# SystemAdministrationBooks

**Note:** *Listed in reverse chronological order; this list is very far from being a complete list of Python system administration books*

## Pro Python System Administration {#Pro_Python_System_Administration}

- Rytis Sileika

ISBN: [1430226056](http://www.amazon.com/exec/obidos/ISBN=1430226056 "ISBN"){.interwiki} Apress, 416 pages (July 15, 2010)

This book explains and shows how to apply Python scripting in practice. In this book, you will find several projects in the categories of network administration, web server administration, and monitoring and database management.

More information: [http://www.sysadminpy.com](http://www.sysadminpy.com){.http}
