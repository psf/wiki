**Jython Books**

1\. [Jython Essentials](http://www.oreilly.com/catalog/jythoness). Samuele Pedroni, Noel Rappin. O\'Reilly and Associates. 2002.

2.\"Jython for Java Programmers.\" Robert Bill. New Riders. 2001.

3\. [Python Programming with the Java Class Libraries: A Tutorial for Building Web and Enterprise Applications](http://www.aw-bc.com/catalog/academic/product/0,1144,0201616165,00.html). Richard Hightower. Addison-Wesley Publishing. 2002.

4\. [An Introduction to Computing and Programming in Python: A Multimedia Approach](http://vig.prenhall.com/catalog/academic/product/0,1144,0131176552,00.html). Mark Guzdial. Prentice Hall. 2004.

5\. [WebSphere Application Server Administration Using Jython](http://www.ibmpressbooks.com/bookstore/product.asp?isbn=9780137009527) Robert A. Gibson, Arthur Kevin [McGrath](./McGrath.html), Noel J. Bergman. IBM Press. 2009.

6\. [Scientific Data Analysis using Jython scripting and Java](http://www.springer.com/computer/book/978-1-84996-286-5) S.V.Chekanov, 2010 (Springer, 2010, 440 p. ISBN 978-1-84996-286-5)

7\. [The Definitive Guide to Jython (Open Source Version)](http://jythonpodcast.hostjava.net/jythonbook/index.html) Josh Juneau, Jim Baker, Victor Ng, Leo Soto, Frank Wierzbicki, 2009/2010

- *[Printed by Apress in Feb 2010](http://www.apress.com/book/view/9781430225270)*
