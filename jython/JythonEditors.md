# Jython Tools {#Jython_Tools}

Coverage of Tools, editors and IDE\'s that support Jython or at least Python to some degree or another. Some of these may have better support for Jython then others. As with most things [YMMV](http://www.urbandictionary.com/define.php?term=YMMV){.http} and all of these can be tried for free.

- ![/!\\](/wiki/modernized/img/alert.png "/!\"){height="16" width="16"} *Please dont hesitate if you have anything to add to this list. Thanks*

**Editors**

- [PyCharm](http://www.jetbrains.com/pycharm/faq/index.html){.http} - is an IDE (Integrated Development Environment) built on top of the IntelliJ Platform and tailored for Python development.

- [Python Editors](http://wiki.python.org/moin/PythonEditors){.http} from the Python Wiki

- [Pydev](http://pydev.org/){.http} - If you use Eclipse, Pydev is awesome.

- [npython](https://nbpython.dev.java.net/){.https} - Netbeans support. for those of us that detest Eclipse.

- [Komodo Edit](http://www.activestate.com/Products/komodo_ide/komodo_edit.mhtml){.http} - ActiveState\'s Free and open source (see [OpenKomodo](http://www.openkomodo.com/){.http}) editor

- [Komodo IDE](http://www.activestate.com/Products/komodo_ide/index.mhtml){.http} - ActiveState\'s Python IDE (commercial)

- [jHepWork](http://jhepwork.sourceforge.net/){.http} - Jython IDE with integrated Jython shell.

- [jEdit](http://www.jedit.org/){.http} - jEdit is a programmer\'s text editor which has a many plugins

  - [JythonInterpreter](http://plugins.jedit.org/plugins/?JythonInterpreter){.http} - provides support for writing jEdit macros in Python.

  - [JPyDbg](http://jpydbg.sourceforge.net/){.http}- for jEdit and Netbeans

- [Python Monkey](http://code.google.com/p/jrfonseca/wiki/PythonMonkey){.http} - adds support for Python language to the Eclipse Monkey plugin, using Jython.
