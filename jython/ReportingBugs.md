# Reporting Bugs

::: {#viewing-and-submitting-a-bugs .section}
### Viewing and submitting a bugs

The Jython project uses a roundup based bug tracker. Go to [http://bugs.jython.org](http://bugs.jython.org){.http .reference .external} for all bug management needs.
:::

::: {#using-the-bug-tracker .section}
### Using the bug tracker

By default, you will see the list of all Open bugs. You can change which bugs you\'re viewing through the [search](http://bugs.jython.org/issue?@template=search&status=1){.http .reference .external} system. To submit a bug, [create an account](http://bugs.jython.org/user?@template=register){.http .reference .external}, login and go to the [issue creation](http://bugs.jython.org/issue?@template=item){.http .reference .external} page
:::
